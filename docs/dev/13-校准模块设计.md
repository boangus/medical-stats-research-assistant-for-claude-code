# 校准模块设计 — Calibration Framework

> **目录**: `shared/calibration/` | **命令**: `/msra-calibrate` | **版本**: v0.9.2 | **更新日期**: 2026-06-23

---

## 1. 模块概述

校准模块为 MSRA Pipeline 提供分析结果的校准验证能力，确保统计指标的准确性和一致性。

### 1.1 角色定位

- **定位**: 分析结果校准与验证
- **输入**: 分析执行结果 (Stage 3 输出) + 黄金标准数据
- **输出**: 校准报告（偏差/精密度/准确度）

### 1.2 校准场景

| 场景 | 说明 | 触发方式 |
|------|------|---------|
| 指标校准 | 验证统计指标（OR/HR/β）的计算准确性 | `/msra-calibrate` |
| 黄金标准比对 | 与已知正确结果比对 | 自动（有黄金标准时） |
| 跨引擎一致性 | R vs Python 结果一致性 | 语言切换时 |

## 2. 校准框架

### 2.1 校准流程

```
分析结果
    │
    ▼
加载黄金标准 (gold_standard_example.csv)
    │
    ▼
逐指标比对
    │  ├─ 偏差 (Bias) = 测量值 - 黄金标准
    │  ├─ 精密度 (Precision) = SD(测量值)
    │  └─ 准确度 (Accuracy) = √(Bias² + Variance)
    ▼
生成校准报告
    │
    ├─ 通过 → 继续
    └─ 未通过 → 标记偏差，建议排查
```

### 2.2 可接受偏差范围

| 指标类型 | 可接受偏差 | 说明 |
|----------|-----------|------|
| 点估计 (OR/HR/β) | ±0.005 | 与黄金标准比较 |
| 置信区间 | ±0.01 | CI 边界偏差 |
| P 值 | 结论性一致 | p<0.05 vs p≥0.05 必须一致 |
| AIC/BIC | ±1.0 | 模型选择指标 |

## 3. 校准运行器

### 3.1 Python 版本

**文件**: `shared/calibration/calibration_runner.py`

```python
from calibration_runner import CalibrationRunner

runner = CalibrationRunner(
    gold_standard_path="shared/calibration/gold_standard_example.csv",
    results_path="MSRA/reports/analysis_results.json"
)

report = runner.run()
print(report.bias_summary)
print(report.is_calibrated)  # True/False
```

### 3.2 R 版本

**文件**: `shared/calibration/calibration_runner.R`

```r
source("shared/calibration/calibration_runner.R")

report <- run_calibration(
  gold_standard = "shared/calibration/gold_standard_example.csv",
  results = "MSRA/reports/analysis_results.rds"
)

print(report$bias_summary)
print(report$is_calibrated)
```

## 4. 黄金标准数据

**文件**: `shared/calibration/gold_standard_example.csv`

黄金标准数据用于验证分析代码的正确性：
- 包含已知正确结果的基准数据集
- 覆盖主要统计方法（t 检验、回归、生存分析等）
- 可扩展：用户可添加自己的黄金标准

### 4.1 使用方式

1. 分析代码首次运行时，与黄金标准比对
2. 偏差在可接受范围内 → 通过
3. 偏差超出范围 → 标记并建议排查

---

## 5. 集成到 Pipeline

校准模块可作为 Stage 3 后的独立验证步骤：

```
Stage 3 分析执行完成
    │
    ▼
校准验证（可选）
    │  ├─ 有黄金标准 → 自动校准
    │  └─ 无黄金标准 → 跳过
    ▼
Stage 3.5 结果质量门闸
```

校准结果记录到 Material Passport 的 `calibration_results` 字段。
