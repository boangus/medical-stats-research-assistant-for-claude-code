# 32. 公共卫生方法模块规划

> 创建时间: 2026-06-27 | 来源: R12公共卫生研究员角色评估 | 状态: 规划中

---

## 1. 背景

R12公共卫生研究员角色评估（综合6.8/10）识别出以下关键差距：

| 维度 | 当前状态 | 目标状态 |
|------|---------|---------|
| 空间分析 | 空白 | Moran's I + GWR + 空间聚类 |
| 传染病建模 | 空白 | SIR/SEIR + Rt估计 |
| 疾病监测 | 通用异常检测 | Farrington/Noufaily算法 |
| 复杂调查 | 仅IPTW权重 | 完整svydesign(strata/cluster/fpc) |
| 时间序列 | 仅CUSUM | ARIMA + ITS + 变点检测 |
| 健康公平 | schema占位 | SII/RII/集中指数/PAF |

**亮点**: Meta-analysis(9/10)、因果推断(DAG+PS+IPTW+DID+MR)、生存分析完整。

## 3. 依赖管理

在`src/shared/templates/00_config_template.R`中统一管理：

```r
# === 公共卫生方法依赖 ===
# 必选 (Sprint A)
install.packages(c("survey", "AER", "rdrobust"))

# 推荐 (Sprint B)
install.packages(c("sf", "spdep", "deSolve"))
# Windows注意: sf需要系统库支持，建议使用RStudio Package Manager

# 可选 (Sprint C)
install.packages(c("surveillance"))  # 仅疾病监测
# install.packages("GWmodel")  # GWR需要编译
```
