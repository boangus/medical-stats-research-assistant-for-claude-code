# 代码模板库 — R/Python 模板清单

> **版本**: v1.0.0 | **更新日期**: 2026-06-26 | **状态**: 已完成

---

## 1. 概述

MSRA 内置丰富的代码模板库，覆盖数据准备、因果推断、统计分析、图表生成等多个领域。模板分为 R 和 Python 两种语言，供 Exec Runner Agent 在 Stage 3 代码生成时引用。

## 7. 模板使用方式

### 7.1 Exec Runner 引用模板

```
Phase 3: 推断分析
    │
    ├─ 主要分析: Cox 回归
    │   → 引用 shared/templates/causal/propensity_score_template.R
    │
    ├─ 敏感性分析: IPTW
    │   → 引用 shared/templates/causal/04_iptw_template.R
    │
    └─ 图表: KM 曲线
        → 引用 shared/templates/misc/publication_figure_template.py
```

### 7.2 模板管理器

`shared/templates/misc/template_manager.py` 提供统一的模板管理接口：

```python
from shared.templates.misc.template_manager import TemplateManager

tm = TemplateManager()
template = tm.get_template("causal", "propensity_score")
code = template.render(data_path="cleaned.csv", exposure="treatment", outcome="survival")
```
