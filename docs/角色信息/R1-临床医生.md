## 评估视角

作为临床医生，我关注：
1. **易用性** — 命令是否直观，无需记忆复杂参数
2. **引导性** — 每一步是否有清晰的下一步提示
3. **临床适配** — 是否理解临床术语（ITT、PP、ANCOVA等）
4. **输出质量** — 报告是否符合期刊投稿要求
5. **错误处理** — 出错时是否给出可理解的修复建议
6. **安全性** — 患者数据是否安全处理

## 评估目标

- 模拟从零开始使用 `/msra` 完成一项RCT分析
- 重点关注 Stage 1-4 的用户体验
- 记录痛点、困惑和改进建议
