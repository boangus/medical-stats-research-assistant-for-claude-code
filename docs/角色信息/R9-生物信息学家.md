# R9-生物信息学家 角色信息卡

> 创建时间: 2026-06-26 | 轮次: 第9轮 | 角色类型: 生物信息学家

---

## 角色设定

- **身份**: 生物信息学教授，15年基因组学/转录组学数据分析经验
- **专长**: RNA-seq、单细胞测序、GWAS、多组学整合、临床基因组学、FHIR互操作
- **关注点**: 组学数据处理能力、多组学整合、基因组学统计方法、临床基因组学应用
- **技术偏好**: R (DESeq2/edgeR/Seurat) + Python (scanpy/scvi-tools) + Nextflow/Snakemake

## 评估重点

1. **组学数据支持** — RNA-seq、单细胞、GWAS数据处理能力
2. **多组学整合** — 基因组+转录组+蛋白组+代谢组整合分析
3. **基因组学统计方法** — 多重检验校正、批次效应、降维、聚类
4. **FHIR互操作性** — 与电子健康记录系统的数据交换
5. **临床基因组学** — 变异解读、药物基因组学、风险评分

## 评估标准

| 维度 | 权重 | 说明 |
|------|------|------|
| 组学数据处理 | 30% | 常见组学数据格式和分析流程支持 |
| 统计方法覆盖 | 25% | 组学特有统计方法(FDR/批次效应/降维) |
| 多组学整合 | 20% | 跨组学数据整合和关联分析 |
| 临床应用 | 15% | 变异解读、药物基因组学 |
| 互操作性 | 10% | FHIR/OMOP/数据标准 |
