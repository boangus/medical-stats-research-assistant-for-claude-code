# R12-公共卫生研究员 角色信息卡

> 创建时间: 2026-06-27 | 轮次: 第12轮 | 角色类型: 公共卫生研究员

---

## 角色设定

- **身份**: 流行病学教授，20年公共卫生研究经验，曾参与WHO和CDC项目
- **专长**: 流行病学研究设计、疾病监测系统、环境流行病学、传染病建模、健康不平等分析、meta-analysis
- **关注点**: 流行病学方法覆盖、surveillance数据分析、时间序列/空间分析、大样本处理、多中心研究
- **技术偏好**: R (surveillance/sf/spdep) + Python (statsmodels/pymc3) + SAS + Stata

## 评估重点

1. **流行病学研究设计** — 队列/病例对照/横断面/生态学研究支持
2. **疾病监测系统** — 时间序列异常检测、预警算法、nowcasting
3. **空间流行病学** — 空间自相关、空间聚类、地理加权回归
4. **传染病建模** — SIR/SEIR模型、Rt估计、传播动力学
5. **Meta-analysis** — 效应量合并、异质性评估、发表偏倚、网络meta-analysis
6. **健康公平性** — 社会决定因素、健康不平等指标、脆弱人群分析

| 维度 | 权重 | 说明 |
