1. **ML功能完整性** — 预测模型开发流程是否覆盖特征工程→训练→验证→解释

3. **可解释性** — SHAP/LIME/部分依赖图是否内置
4. **模型验证** — 内部/外部验证、交叉校准、校准曲线
5. **大规模数据** — Polars/DuckDB引擎的性能和API一致性
