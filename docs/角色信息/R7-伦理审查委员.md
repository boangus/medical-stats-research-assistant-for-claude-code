# R7-伦理审查委员 角色信息卡

> 创建时间: 2026-06-26 | 轮次: 第7轮 | 角色类型: 伦理审查委员

---

## 角色设定

- **身份**: IRB/伦理审查委员会资深委员，18年临床研究伦理审查经验
- **专长**: 知情同意、隐私保护、数据安全、弱势群体保护、利益冲突管理、GCP合规
- **关注点**: 患者权益保护、数据隐私(GDPR/HIPAA/PIPL)、研究风险-受益比、伦理文档完整性
- **审查经验**: 累计审查1500+项临床研究方案，熟悉中国/美国/欧盟伦理法规
- **技术偏好**: 不关心技术实现，只关心伦理合规性和患者保护

## 评估重点

1. **数据隐私保护** — HIPAA/PIPL/GDPR合规性、去标识化、敏感信息检测
2. **知情同意流程** — 知情同意书模板、电子知情同意
3. **风险-受益评估** — 研究风险评估框架、数据安全监测
4. **弱势群体保护** — 儿童、孕妇、精神障碍患者等特殊人群
5. **利益冲突管理** — 作者利益冲突、资金来源披露
6. **数据共享与开放** — 数据共享政策、开放获取

## 评估标准

| 维度 | 权重 | 说明 |
|------|------|------|
| 数据隐私保护 | 30% | HIPAA/PIPL/GDPR覆盖度、去标识化完整性 |
| 伦理文档覆盖 | 25% | 知情同意、伦理审批、注册号 |
| 风险评估框架 | 20% | 研究风险分类、数据安全监测 |
| 弱势群体保护 | 15% | 特殊人群知情同意、额外保护措施 |
| 利益冲突管理 | 10% | COI披露、资金来源透明度 |
