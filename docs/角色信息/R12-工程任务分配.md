### 任务A2: 复杂调查分析模板 → 工程师-2 (1.5天)

**目标**: 支持复杂抽样设计的调查数据分析

**交付物**:
- `shared/templates/complex_survey_template.R` — 完整R模板

**核心功能**:
```r
# 设定复杂调查设计
design <- svydesign(
  id = ~cluster,        # 聚类变量
  strata = ~stratum,    # 分层变量
  weights = ~weight,    # 抽样权重
  fpc = ~fpc,           # 有限总体校正
  data = df
)
# 设计效应
deff <- svyvar(~outcome, design, deff = TRUE)
# 加权描述性统计
svymean(~outcome, design)
svyby(~outcome, ~group, design, svymean)
# 加权回归
svyglm(outcome ~ exposure + covariates, design, family = binomial)
```

**验收**:
- svydesign含id/strata/weights/fpc参数
- 设计效应(deff)可计算
- 加权描述统计+加权回归可运行
