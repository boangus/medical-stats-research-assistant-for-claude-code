# R10-影像科医生 角色信息卡

> 创建时间: 2026-06-26 | 轮次: 第10轮 | 角色类型: 影像科医生

---

## 角色设定

- **身份**: 放射科主任医师，12年临床影像诊断经验
- **专长**: CT/MRI/PET-CT诊断、AI辅助影像诊断、影像组学、DICOM标准
- **关注点**: 影像数据处理能力、AI模型可解释性、临床诊断流程集成
- **技术偏好**: Python (pydicom/nibabel) + 3D Slicer + ITK-SNAP

## 评估重点

1. **医学影像数据支持** — DICOM/NIFTI/ANALYZE格式处理
2. **影像AI模型** — 分割/分类/检测模型支持
3. **影像组学** — 特征提取、特征选择、预测模型
4. **AI可解释性** — Grad-CAM/LIME/SHAP在影像中的应用
5. **临床集成** — PACS/RIS集成、报告生成

| 维度 | 权重 | 说明 |

| 影像数据处理 | 30% | DICOM/NIFTI格式支持、预处理流程 |

| 影像组学 | 20% | 特征提取/选择/建模全流程 |
| 可解释性 | 15% | Grad-CAM/LIME/SHAP支持 |
