# MSRA Plan

路由到 `skills/analysis-plan/SKILL.md`，传入用户参数 `$ARGUMENTS`。

## 参数解析

- `<data>` — 清洗后数据文件
- `--study <type>` — 研究类型（RCT/observational/diagnostic）
- `--quick` — 快速模式

估计目标定义（ICH E9(R1)）、方法选择决策树、SAP 制定与审查、研究类型分支逻辑均在 `skills/analysis-plan/SKILL.md` 中定义。
