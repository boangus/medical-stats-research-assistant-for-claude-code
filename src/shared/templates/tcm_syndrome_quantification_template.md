## 2. 证候疗效判定标准

### 2.1 计算公式

```
疗效指数 = (治疗前积分 - 治疗后积分) / 治疗前积分 × 100%
```

### 2.2 判定标准

| 疗效等级 | 标准 | 编码 |
|---------|------|------|
| 痊愈 | 证候积分减少 ≥ 95% | CR |
| 显效 | 证候积分减少 70%-94% | PR |
| 有效 | 证候积分减少 30%-69% | MR |
| 无效 | 证候积分减少 < 30% | NR |

### 2.3 总有效率计算

```
总有效率 = (痊愈 + 显效 + 有效) / 总例数 × 100%
显效率 = (痊愈 + 显效) / 总例数 × 100%
```

## 3. 统计分析方法

### 3.1 证候积分分析

```r
# R 代码
library(gtsummary)

# 组间比较（积分变化量）
# 正态: t检验 / 非正态: Wilcoxon
wilcox.test(post_score - pre_score ~ group, data = df)

# 效应量
effectsize::cohens_d(score_change ~ group, data = df)

# 重复测量分析
lme4::lmer(score ~ group * time + (1 | subject_id), data = df_long)
```

### 3.2 疗效等级分析

```r
# R 代码

# 疗效等级的组间比较
chisq.test(table(df$efficacy, df$group))

# 有序Logistic回归
library(MASS)
polr(efficacy ~ group + age + sex, data = df)

# 总有效率比较
prop.test(
  x = c(sum(df$group == 1 & df$effective), sum(df$group == 0 & df$effective)),
  n = c(sum(df$group == 1), sum(df$group == 0))
)
```

### 3.3 信效度评价

```r
# Cronbach's α（内部一致性信度）
library(psych)
alpha(df[, symptom_cols])

# ICC（重测信度）
library(irr)
icc(test_retest_data, model = "twoway", type = "agreement")

# Kappa（评估者间一致性）
library(irr)
kappa2(raters_data, weight = "squared")  # 加权Kappa（有序数据）
```

## 5. 注意事项

| 注意事项 | 说明 |
|---------|------|
| 评分表来源 | 应引用已发表的标准化量表或自建量表需报告信效度 |
| 主次症权重 | 主症权重应高于次症（2倍以上） |
| 判定标准 | 30%/70%/95% 切点需在方案中预先设定 |
| 舌脉处理 | 舌脉通常作为辅助/探索性指标，不计入总分 |
| 个体化证候 | 中医辨证个体化特点需要分层分析或敏感性分析 |
