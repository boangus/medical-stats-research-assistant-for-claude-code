## 3. 计算方法与公式

### 3.1 计算方法选择

| 结局类型 | 比较类型 | 计算方法 | 选择理由 |
|---------|---------|---------|---------|
| [{连续型}] | [{两独立组比较}] | [{两样本t检验公式}] | [{参数假设满足}] |
| [{二分类}] | [{两率比较}] | [{卡方检验 / Fisher精确检验}] | [{事件率适用}] |
| [{生存}] | [{两组比较}] | [{Log-rank检验 / Cox模型}] | [{含删失数据}] |
| [{重复测量}] | [{两组比较}] | [{MMRM / GEE}] | [{考虑相关性}] |

### 3.2 计算公式

#### 3.2.1 连续型结局（两独立组t检验）

```
n_per_group = 2 × (Z_{1-α/2} + Z_{1-β})² × σ² / Δ²

其中：
  Z_{1-α/2} = 标准正态分位数（双侧α=0.05时为1.96）
  Z_{1-β} = 标准正态分位数（Power=0.80时为0.84）
  σ = 合并标准差
  Δ = 预期均值差（μ_干预 - μ_对照）

总样本量 N = 2 × n_per_group / (1 - 脱落率)
```

#### 3.2.2 二分类结局（两率比较）

```
n_per_group = (Z_{1-α/2} × sqrt(2 × p_bar × (1-p_bar)) + Z_{1-β} × sqrt(p_1×(1-p_1) + p_2×(1-p_2)))² / (p_1 - p_2)²

其中：
  p_1 = 对照组事件率
  p_2 = 干预组事件率
  p_bar = (p_1 + p_2) / 2

总样本量 N = 2 × n_per_group / (1 - 脱落率)
```

#### 3.2.3 生存结局（Log-rank检验）

```
事件数 D = (Z_{1-α/2} + Z_{1-β})² / [p_1 × p_2 × (ln(HR))²]

其中：
  p_1, p_2 = 两组样本比例
  HR = 风险比

总样本量 N = D / (整体事件率 × (1 - 脱落率))
```

#### 3.2.4 非劣效性检验

```
n_per_group = 2 × (Z_{1-α} + Z_{1-β})² × σ² / (Δ - Δ_NI)²

其中：
  Δ = 预期效应量（通常假设为0或基于既往数据）
  Δ_NI = 非劣效界值
  注意：非劣效通常使用单侧α
```

### 3.3 调整因素

| 调整因素 | 调整方法 | 调整系数 | 调整后样本量 |
|---------|---------|---------|------------|
| 脱落/失访 | N_adj = N_raw / (1 - r_dropout) | [{如：÷0.80}] | [{调整后值}] |
| 多重比较 | [{如：Bonferroni α调整}] | [{如：α' = α/k}] | [{调整后值}] |
| 不等分配 | n_adj = n × (1 + r)² / (4r)（r为分配比） | [{如：r=2时×1.125}] | [{调整后值}] |
| 整群设计 | N_adj = N × DEFF（DEFF = 1 + (m-1)×ICC） | [{如：DEFF=1.98}] | [{调整后值}] |

# R计算代码示例（两样本t检验）
library(pwr)
result <- pwr.t.test(
  d = 0.5,          # Cohen's d 效应量
  sig.level = 0.05,  # α
  power = 0.80,      # 把握度
  type = "two.sample",
  alternative = "two.sided"
)
print(result)

# 脱落率调整
dropout_rate <- 0.20
n_adj <- ceiling(result$n / (1 - dropout_rate))
cat("每组调整后样本量:", n_adj, "\n")
cat("总样本量:", 2 * n_adj, "\n")
```

# Python计算代码示例（两样本t检验）
from statsmodels.stats.power import TTestIndPower
from statsmodels.stats.power import tt_ind_solve_power

effect_size = 0.5  # Cohen's d
alpha = 0.05
power = 0.80

n_per_group = tt_ind_solve_power(
    effect_size=effect_size,
    alpha=alpha,
    power=power,
    ratio=1.0,
    alternative='two-sided'
)
print(f"每组样本量: {int(n_per_group) + 1}")

# 脱落率调整
dropout_rate = 0.20
n_adj = int(n_per_group / (1 - dropout_rate)) + 1
print(f"调整后每组样本量: {n_adj}")
print(f"总样本量: {2 * n_adj}")
```
