# CONSORT 报告规范检查清单

> 适用于随机对照试验（RCT）
> 基于 **CONSORT 2025 声明**（Hopewell et al., Lancet/BMJ/JAMA/Nature Med/PLOS Med 2025, PMC11996237）
> 30 项官方清单结构 + 整合 CONSORT 扩展（Harms、TIDieR、非药物干预）
> 更新日期: 2026-06-14

> **IRON RULES**:
> - 本清单严格对齐 CONSORT 2025 官方 30 项编号，不得自创编号体系
> - 7 个新增项（Item 8/12a/12b/13b/24/26/30）以 **🆕 2025** 标注
> - **Item 8 (PPI) 和 Item 24 (干预实施 TIDieR 整合) 是 2025 版最实质性的新增**，缺失即判定报告不合规
> - 参考: resources/anti-patterns/medical_stats_anti_patterns.md（D1/D2/D3）

---

## Title and Abstract

### Item 1. 标题和摘要
- [ ] 标题中标识为随机试验
- [ ] 结构化摘要（背景/目的/方法/结果/结论），报告主要结局效应量+95%CI+精确 P 值

## Introduction

### Item 2. 背景和目的
- [ ] 科学背景与原理
- [ ] 明确研究目的和预设假设

## Methods

### Item 3. 试验设计
- [ ] 描述试验设计（平行/交叉/析因/集群等）
- [ ] 说明随机化比例
- [ ] 试验开始后的重要变更（设计、剔除标准、结局）及理由

### Item 4. 随机化前的停止规则
- [ ] 🆕 **2025** 描述提前停止试验的规则（如有）：决策依据（期中分析/无效性/危害性）和独立委员会

### Item 5. 参与者
- [ ] 合格标准（纳入/排除）
- [ ] 招募来源和设置（地点/中心类型）
- [ ] 🆕 **2025** **参与者多样性特征**：年龄、性别、种族/民族、社会经济状况的目标分布
- [ ] 🆕 **2025** 为促进代表性不足人群纳入采取的措施（如有）

### Item 6. 干预措施
- [ ] 详细描述各组干预措施（内容、剂量、频次、时程、实施者资质）
- [ ] 干预措施开发依据（理论/证据基础）

### Item 7. 结局指标
- [ ] 明确定义主要和次要结局（含测量方法、时间点、评估者）
- [ ] 任何衍生/复合指标的定义
- [ ] 结局变更（若有）及理由

### Item 8. 患者和公众参与（PPI） 🆕 **2025**
- [ ] **患者/公众在研究设计、实施、报告或解读中的参与方式**
- [ ] PPI 对研究的影响（如结局选择、招募策略、知情同意书措辞）
- [ ] 无 PPI 时须明确声明并说明理由

### Item 9. 样本量
- [ ] 样本量计算方法（效应量、α、Power、脱落假设）
- [ ] 期中分析与样本量重估规则（如有）

### Item 10. 随机化序列生成
- [ ] 随机序列生成方法（简单/分层/区组/最小化）
- [ ] 分层因素和区组大小

### Item 11. 分配隐藏
- [ ] 分配隐藏机制（顺序编号容器/中心电话/网页系统）
- [ ] 实施随机化的人员及其角色

### Item 12a. 实施
- [ ] 谁生成随机序列、谁入组参与者、谁分配参与者

### Item 12b. 干预实施 🆕 **2025**
- [ ] **干预实施者（who）的资质/培训标准**
- [ ] **实施地点和环境的合格标准**
- [ ] **伴随治疗（concomitant care）的控制或记录**（整合 TIDieR）

### Item 13a. 盲法
- [ ] 盲法实施：谁被盲（参与者/实施者/结局评估者/统计分析师/数据分析者）
- [ ] 安慰剂/模拟干预（如有）

### Item 13b. 非药物干预的盲法 🆕 **2025**
- [ ] **对参与者、干预实施者、结局评估者的盲化策略**（针对非药物干预特有挑战）
- [ ] 盲法破除记录

### Item 14. 统计方法
- [ ] 主要/次要结局的统计方法
- [ ] 亚组分析和敏感性分析
- [ ] 缺失数据处理方法
- [ ] ITT/PP/mITT 分析策略
- [ ] 🆕 **2025** 提供统计分析计划（SAP）的获取途径或引用

## Results

### Item 15. 参与者流程
- [ ] 提供 CONSORT 流程图（入组→分配→随访→分析）
- [ ] 报告各阶段人数及失访/排除原因
- [ ] 随机化后偏离原定方案的记录

### Item 16. 募集
- [ ] 招募期和随访期
- [ ] 试验结束/停止的原因（如有）

### Item 17. 基线数据
- [ ] 各组基线特征表（人口学+临床特征）
- [ ] 🆕 **2025** 按预定义亚组（年龄/性别/种族）报告基线特征以评估多样性

### Item 18. 分析人数
- [ ] 各组进入主要分析的人数（分母）
- [ ] ITT/mITT/PP 人群的定义和差异

### Item 19. 结局和估计
- [ ] 主要结局：效应量 + 95%CI + 精确 P 值（不写 P<0.05）
- [ ] 次要结局：效应量 + 95%CI
- [ ] 绝对和相对效应量
- [ ] 其他分析（亚组/敏感性）结果

### Item 20. 辅助分析
- [ ] 亚组分析与预设方案的对应（事后分析须标注）
- [ ] 多重比较的处理

### Item 21a. 危害（Harms 总体）
- [ ] 报告各组所有严重不良事件（SAE）和常见 AE（≥5%）
- [ ] 按系统器官分类（SOC，MedDRA）报告
- [ ] 🆕 **2025** **Harms 作为独立的主要/次要结局**时须完整报告（整合 CONSORT Harms 扩展）

### Item 21b. 危害（细节） 🆕 **2025**
- [ ] AE 收集方法（主动/被动、随访时点）
- [ ] 因 AE 退出/剂量调整/死亡的人数
- [ ] AE 的因果评估方法（研究者判定/独立委员会）

### Item 22. 干预实施情况（Delivery） 🆕 **2025**
- [ ] 实际实施的干预与方案的符合度（fidelity）
- [ ] 各组接受完整干预的人数比例
- [ ] 实施偏差（protocol deviations）的类型和数量

## Discussion

### Item 23. 局限性
- [ ] 试验局限性（偏倚来源、测量误差、多重比较）
- [ ] 外部效度（可推广性）的讨论

### Item 24. 临床意义 vs 统计显著性 🆕 **2025**
- [ ] **结果解读须区分临床意义与统计显著性**（即使 p<0.05，效应量很小时须谨慎）
- [ ] 讨论结果对不同人群亚组的适用性
- [ ] 与既往证据的一致性

## Other information

### Item 25. 注册
- [ ] 试验注册号和注册平台
- [ ] 🆕 **2025** 方案的公开获取途径（链接）

### Item 26. 方案 🆕 **2025**
- [ ] 完整试验方案的获取途径（公开链接/补充材料/数据仓）
- [ ] 方案与实际执行的偏差及原因（protocol deviations 清单）

### Item 27. 数据共享 🆕 **2025**
- [ ] **个体参与者数据（IPD）的获取途径和条件**（Dryad/Zenodo/Vivli 链接或声明"不共享"+理由）
- [ ] 完整数据集还是部分数据集；何时可用、可使用多久
- [ ] 分析代码和统计程序的获取途径

### Item 28. 资金
- [ ] 资金来源和资助者角色（是否参与设计/收集/分析/报告）
- [ ] 商业赞助的独立性声明

### Item 29. 利益冲突
- [ ] 所有作者的利益冲突声明（财务/雇佣/咨询/专利关系）
- [ ] 独立于商业利益的数据分析声明

### Item 30. AI/工具使用披露 🆕 **2025**
- [ ] **声明是否使用 AI 工具（LLM/ML）参与研究设计、数据分析或论文撰写**
- [ ] 若使用，说明具体用途、工具名称、版本和人类监督方式
- [ ] 使用 MSRA 等 AI 辅助工具须在此项披露

## CONSORT 2025 七项实质性新增（速查）

| Item | 类型 | 新增内容 | 缺失判定 |
|------|------|---------|---------|
| 4 | Methods | 停止规则 | 不适用（无提前停止时可标注 N/A） |
| 8 | Methods | **PPI（患者与公众参与）** | **核心项缺失→不合规** |
| 12b | Methods | 干预实施者资质+伴随治疗 | 建议补充 |
| 13b | Methods | 非药物干预盲法 | 非药物试验必查 |
| 21b | Results | Harms 收集方法+因果评估 | 安全性试验必查 |
| 22 | Results | 干预实施符合度（TIDieR 整合） | 实施性研究必查 |
| 24 | Discussion | 临床意义 vs 统计显著性 | **核心项缺失→不合规** |
| 26 | Other | 方案获取途径+偏差清单 | 建议补充 |
| 27 | Other | IPD/代码共享声明 | 建议补充 |
| 30 | Other | AI 工具使用披露 | 使用 MSRA 时必查 |

## 关键提示

1. **报告必须使用 30 项官方编号**（Item 1–30），不得用自创编号体系
2. **统计量一致性**: P 值、效应量、95%CI 必须相互一致 — 参考 `src/shared/reporting-guidelines/statcheck_rules.md` 自动校验
3. **主要结局**: 严格报告效应量 + 95%CI + 精确 P 值，禁止 `P<0.05` 二元表述（参考 anti-patterns D1）
4. **流程图**: CONSORT 流程图是强制要求
5. **核心必查项**: Item 8 (PPI)、Item 24 (临床意义)、Item 30 (AI 披露) 缺失即判定不合规
6. **多样性报告**: 参考 FDA Diversity Action Plan 2024，按预定义亚组报告基线和结局（Item 5/17）
7. **资料来源**: Hopewell S, et al. CONSORT 2025 statement: updated guideline for reporting randomised trials. BMJ 2025;389:2024-081123. PMC11996237.
