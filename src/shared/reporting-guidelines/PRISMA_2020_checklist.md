# PRISMA 2020 系统综述报告规范检查清单

> 适用于系统综述和 Meta 分析（包括定量和定性综合）
> 基于 **PRISMA 2020 声明**（Page et al., BMJ 2021; PMC7932247）
> 参考: [EQUATOR Network](https://www.equator-network.org/reporting-guidelines/prisma/) | [prisma-statement.org](http://www.prisma-statement.org/)
> 27 项官方清单结构 + PRISMA 2020 流程图模板
> 更新日期: 2026-06-15

> **IRON RULES**:
> - 本清单严格对齐 PRISMA 2020 官方 27 项编号，不得自创编号体系
> - 适用于常规系统综述/Meta 分析；网络 Meta 分析请用 `PRISMA_NMA_checklist.md`
> - 参考：resources/risk-of-bias/ — 纳入研究的偏倚评估工具（RoB 2/ROBINS-I/PROBAST/QUADAS-2）
> - 参考：resources/risk-of-bias/GRADE_framework.md — 证据确定性评估

---

## Title

### Item 1. 标题
- [ ] 标题中标识为系统综述、Meta 分析或两者
- [ ] 如为网络 Meta 分析，使用 `PRISMA_NMA_checklist.md`

---

## Abstract

### Item 2. 摘要
- [ ] 结构化摘要包含：背景、目的、数据来源、研究纳入标准、主要结局、数据综合方法、结果、结论
- [ ] 报告纳入研究数量和参与者总数
- [ ] 报告主要结局的效应量和置信区间
- [ ] 注册号（如 PROSPERO）

---

## Introduction

### Item 3. 理论基础
- [ ] 说明综述的理论依据和目的
- [ ] 说明综述问题与已有综述的关系

### Item 4. 目的
- [ ] 明确综述的研究问题（PICO/PECO 框架）
- [ ] 说明主要和次要结局
- [ ] 说明纳入标准（研究设计、人群、干预、对照、结局、时间、设置）

---

## Methods

### Item 5. 方案和注册
- [ ] 注册信息（PROSPERO 或其他平台）
- [ ] 注册号和 URL
- [ ] 方案可获取性（如有）
- [ ] 方案变更说明（如有）

### Item 6. 纳入标准
- [ ] 详细纳入标准（PICO/PECO + 研究设计 + 语言 + 时间 + 设置）
- [ ] 说明是否排除特定研究（如动物研究、会议摘要）

### Item 7. 信息来源
- [ ] 列出所有数据库（如 PubMed、Embase、Cochrane Library、Web of Science）
- [ ] 说明其他信息来源（试验注册平台、灰色文献、手工检索、联系作者）
- [ ] 最后检索日期

### Item 8. 检索策略
- [ ] 至少一个数据库的完整检索策略（附录）
- [ ] 检索策略是否包含所有 PICO 要素
- [ ] 是否使用了过滤器（如语言、时间）
- [ ] 检索策略是否经过同行评审（如 PRESS 清单）

### Item 9. 选择过程
- [ ] 说明研究选择方法（独立筛选、解决分歧机制）
- [ ] 说明筛选阶段（标题/摘要 → 全文）
- [ ] 说明是否使用自动化工具辅助筛选

### Item 10. 数据提取
- [ ] 说明数据提取方法（独立提取、解决分歧机制）
- [ ] 说明提取的变量列表
- [ ] 说明是否联系原作者获取缺失数据

### Item 11. 偏倚风险评估
- [ ] 说明偏倚风险评估工具（如 RoB 2、ROBINS-I、QUADAS-2、PROBAST）
- [ ] 说明评估方法（独立评估、解决分歧机制）
- [ ] 说明如何将偏倚风险纳入结果综合（如分层分析、敏感性分析）

> 参考：resources/risk-of-bias/ — 完整的偏倚评估工具库

### Item 12. 效应量指标
- [ ] 说明主要效应量指标（如 RR、OR、MD、SMD）
- [ ] 说明效应量计算方法
- [ ] 说明方向性（如 OR>1 代表什么）

### Item 13. 数据综合方法
- [ ] 说明 Meta 分析方法（固定效应/随机效应模型）
- [ ] 说明异质性评估方法（I²、Q 检验、τ²）
- [ ] 说明亚组分析和 Meta 回归的预设变量
- [ ] 说明敏感性分析的预设方案
- [ ] 说明如何处理缺失数据（如 imputation、complete case）
- [ ] 说明是否进行了发表偏倚评估（如漏斗图、Egger 检验）

### Item 14. 报告偏倚评估
- [ ] 说明报告偏倚评估方法（如漏斗图不对称性检验）
- [ ] 说明如何处理报告偏倚（如 trim-and-fill）

### Item 15. 确定性评估
- [ ] 说明证据确定性评估方法（如 GRADE）
- [ ] 说明评估单位（逐结局评估）

> 参考：resources/risk-of-bias/GRADE_framework.md — GRADE 证据确定性框架

---

## Results

### Item 16. 研究选择
- [ ] PRISMA 2020 流程图（含识别、筛选、纳入、排除各阶段数量）
- [ ] 排除研究列表及排除理由（附录）
- [ ] 多轮筛选的一致性（如 Cohen's κ）

### Item 17. 研究特征
- [ ] 纳入研究特征表（作者、年份、设计、样本量、干预、对照、结局、随访时间）
- [ ] 纳入研究的偏倚风险汇总表

### Item 18. 偏倚风险评估结果
- [ ] 逐研究偏倚风险评估结果（表格或图形）
- [ ] 按偏倚等级分层的效应量比较

### Item 19. 单个研究结果
- [ ] 逐研究效应量和置信区间（森林图）
- [ ] 每个结局的纳入研究数量

### Item 20. 数据综合结果
- [ ] Meta 分析结果（合并效应量、95% CI、I²、τ²）
- [ ] 亚组分析结果（如有）
- [ ] Meta 回归结果（如有）
- [ ] 敏感性分析结果（如排除高偏倚研究、不同效应模型）

### Item 21. 报告偏倚评估结果
- [ ] 漏斗图（如有 ≥10 个研究）
- [ ] 不对称性检验结果（如 Egger 检验 p 值）

### Item 22. 确定性评估结果
- [ ] GRADE 证据确定性等级（逐结局）
- [ ] 证据概要表（Summary of Findings, SoF）

---

## Discussion

### Item 23. 讨论
- [ ] 主要发现总结
- [ ] 与已有综述/指南的一致性或差异
- [ ] 证据的确定性和局限性（偏倚风险、不一致性、间接性、不精确性）

### Item 24. 局限性
- [ ] 综述层面的局限性（如检索范围、语言限制）
- [ ] 研究层面的局限性（如纳入研究质量）
- [ ] 结局层面的局限性（如证据确定性等级）

### Item 25. 结论和启示
- [ ] 对临床实践/政策的启示
- [ ] 对未来研究的建议
- [ ] 证据缺口的识别

---

## Other

### Item 26. 注册和方案
- [ ] 注册号和 URL（重复 Item 5，确保报告中出现）
- [ ] 方案可获取性
- [ ] 方案变更说明

### Item 27. 资金和利益冲突
- [ ] 综述的资金来源
- [ ] 资金在综述中的作用
- [ ] 作者利益冲突声明
- [ ] 纳入研究的资金和利益冲突（如有）

## 与 PRISMA-NMA 的区别

| 特征 | PRISMA 2020（本清单） | PRISMA-NMA |
|------|----------------------|------------|
| 适用类型 | 常规系统综述/Meta 分析 | 网络 Meta 分析 |
| 核心特点 | 直接比较 A vs B | 多干预间接比较 |
| 清单项数 | 27 项 | 25 项（含 NMA 特有项） |
| 流程图 | 标准 PRISMA 流程图 | 网络证据图 + 流程图 |
| 异质性 | I²/Q/τ² | 一致性模型 + CINeMA |
