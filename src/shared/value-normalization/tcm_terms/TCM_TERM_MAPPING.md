## 映射原则

### 1. 后缀去重

中医术语常见的后缀变体，检测时自动去后缀匹配：

| 原值 | 去后缀后 | 映射至标准术语 |
|------|---------|-------------|
| 心脾两虚 | 心脾两虚 | 心脾两虚证 |
| 心脾两虚证 | 心脾两虚证 | 心脾两虚证 |
| 心脾两虚证型 | 心脾两虚 | 心脾两虚证 |
| 肝郁脾虚候 | 肝郁脾虚 | 肝郁脾虚证 |
| 湿热下注型 | 湿热下注 | 湿热下注证 |

### 2. 术语别名/同义词

同一概念的不同命名方式，映射至首选标准术语：

| 非标准变体 | 标准术语 | 说明 |
|-----------|---------|------|
| 心脾两虚 / 心脾不足 / 心脾俱虚 | 心脾两虚证 | 同一证候的不同表述 |
| 肝气郁结 / 肝郁气滞 / 肝郁 | 肝气郁结证 | 繁简变体 |
| 气滞血瘀 / 血瘀气滞 | 气滞血瘀证 | 语序变体 |

### 3. 复合证候拆分

当单个单元格包含多个证候（如"心脾两虚，痰湿内阻"）时，按 `，`、`、`、`兼` 等分隔符拆分后分别映射，再由用户选择是否保留为复合值或拆分为多列。

## 更新说明

- 当国标版本更新时，替换 `gb_t16751_syndrome.json` 文件即可
- 如需扩展自定义术语，在 `TCM_TERM_MAP` 中添加条目，自动化工具会标记为标准术语库中未收录的"自定义映射"
- 国家中医药管理局发布的《中医药标准体系表（2025年版）》提供了术语分类的顶层框架，可作为字段级分类参考
