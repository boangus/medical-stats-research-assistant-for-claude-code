# SAP 合规检查协议

**Status**: v1.0 (introduced 2026-06-22)
**Purpose**: 确保统计分析严格遵循已批准的统计分析计划（SAP）。

---

## 1. SAP 锁定验证

### 1.1 锁定条件
- 数据库锁定（database lock）已完成
- SAP 已签署并注明日期
- 盲态审核已完成

### 1.2 锁定后变更
- 任何 SAP 偏离必须记录在偏离日志中
- 分析前偏离 vs 分析后偏离的区分
- 敏感性分析覆盖所有重大偏离

## 2. 分析一致性检查

### 2.1 主要分析
- 主要结局指标定义与 SAP 一致
- 统计方法与 SAP 一致
- 样本量计算与 SAP 一致
- 显著性水平与 SAP 一致

### 2.2 次要分析
- 次要结局指标列表与 SAP 一致
- 亚组分析定义与 SAP 一致
- 敏感性分析方案与 SAP 一致

## 3. 多重比较校正

### 3.1 校正方法
- Bonferroni 校正（主要结局）
- Holm-Bonferroni 校正（次要结局）
- FDR 校正（探索性分析）

### 3.2 报告要求
- 每个比较的原始 p 值
- 校正后的 p 值或置信区间
- 多重比较校正方法说明

## 4. 缺失数据处理

### 4.1 处理策略
- 主要分析：完整案例分析或 SAP 指定的插补方法
- 敏感性分析：多种插补方法对比
- 缺失模式分析：MCAR/MAR/MNAR 判定

### 4.2 报告要求
- 缺失率按变量和组别报告
- 插补方法和参数详细说明
- 敏感性分析结果

## 5. 偏离记录

### 5.1 偏离分类
- **重大偏离**: 影响主要结局解释的偏离
- **次要偏离**: 不影响主要结局解释的偏离

### 5.2 偏离报告
- 偏离描述
- 偏离原因
- 对结果的影响评估
- 补救措施
