# 知情同意书模板 (Informed Consent Form, ICF)

> 版本: v1.0.0 | 更新: 2026-06-26
> 适用: 涉及人的医学研究（RCT/观察性/诊断试验/生物样本研究）
> 参考: ICH-GCP E6(R2) §4.8 | 赫尔辛基宣言(2013) | 《涉及人的生物医学研究伦理审查办法》(2016)

---

## 标准版知情同意书

```
[机构名称/伦理委员会名称]
[研究标题]

知情同意书

版本号: [版本] | 版本日期: [YYYY-MM-DD]
伦理批准号: [编号] | 批准日期: [YYYY-MM-DD]


# 一、研究介绍

尊敬的参与者：

您被邀请参加一项医学研究。在您决定是否参加之前，请仔细阅读本知情同意书。如有任何疑问，请随时向研究者提问。

# 二、研究目的

本研究旨在 [用通俗语言描述研究目的，避免专业术语]。

# 三、研究设计与方法

- 研究类型: [RCT/队列研究/病例对照/诊断试验等]
- 研究地点: [医院/机构名称]
- 预计参与人数: [N] 人
- 您的参与时间: 约 [时长]（包括 [N] 次随访，每次约 [时长]）

# 四、研究程序

如果您同意参加，将进行以下步骤：

1. [步骤1描述 — 用通俗语言]
2. [步骤2描述]
3. [步骤3描述]
4. ...

# 五、随机分组（仅RCT适用）

本研究采用随机分组（类似抛硬币的方式），您有 [概率] 的概率被分配到：
- 试验组: [干预措施描述]
- 对照组: [对照措施描述]

您和您的医生均不知道您被分到哪一组（双盲），除非紧急医疗需要。

# 六、风险与不适

参加本研究可能的风险包括：

常见风险:
- [风险1]: 发生率约 [X]%，表现为 [症状]
- [风险2]: 发生率约 [X]%

罕见但严重风险:
- [风险3]: 发生率约 [X]%，如发生将 [处理措施]

研究者将密切监测您的健康状况，如出现任何不适，请立即联系研究者。

# 七、预期受益

直接受益:
- [可能的直接健康受益]

间接受益:
- 您的参与将帮助 [未来患者/医学知识进步]

请注意，研究结果可能对您个人没有直接益处。

# 八、替代治疗方案

您可以选择不参加本研究，替代方案包括：
- [替代方案1]
- [替代方案2]

这些替代方案的利弊已由您的医生与您讨论。

# 九、隐私保护

- 您的个人信息将严格保密
- 研究数据将以编码形式存储，不包含可识别身份的信息
- 数据仅用于本研究目的
- 研究结果发表时不会泄露您的身份
- 数据存储期限: [N] 年，存储地点: [地点]
- [如适用] 数据可能被共享给 [合作机构]，已签署数据使用协议

# 十、自愿参与与退出

- 参加本研究完全自愿
- 您有权在任何时候退出研究，无需说明理由
- 退出研究不会影响您获得正常的医疗服务
- 如您退出，已收集的数据将继续用于研究（除非您要求删除）

# 十一、费用与补偿

- 研究相关检查和治疗费用由 [资助方/研究者] 承担
- [如有] 您将获得 [交通补贴/误工补偿] 共 [金额] 元
- [如有] 如因研究导致健康损害，将获得免费医疗和适当补偿

# 十二、联系方式

如有疑问或出现健康问题，请联系：

研究负责人: [姓名], [职称]
电话: [号码]
邮箱: [邮箱]

伦理委员会: [名称]
电话: [号码]
邮箱: [邮箱]

# 十三、同意声明

我已阅读并理解本知情同意书的内容，我的所有问题已得到满意回答。我自愿同意参加本研究。

参与者签名: ________________  日期: ____________
（请用正楷书写姓名）

研究者签名: ________________  日期: ____________

见证人签名（如适用）: ________________  日期: ____________
```

## 儿童版简化知情同意书（7-17岁）

```
[研究标题] — 儿童知情同意书

# 亲爱的小伙伴：

我们要邀请你参加一个关于 [主题] 的研究。

# 这个研究是做什么的？

[用儿童能理解的语言描述，配合图片]

# 你需要做什么？

1. [步骤1 — 简单描述]
2. [步骤2]

# 会疼吗/有危险吗？

[诚实描述不适，但不过度恐吓]

# 有什么好处？

[用儿童能理解的语言描述]

# 你可以随时说"不"

- 你不想参加就不参加，没关系
- 你参加了也可以随时退出
- 你的爸爸妈妈/老师不会因此生气

# 你的信息会保密吗？

你的名字不会出现在任何报告里，只有研究的叔叔阿姨知道你参加了。

# 你同意吗？

我读了上面的内容，我 [同意/不同意] 参加这个研究。

小朋友签名: ________________  日期: ____________

# 家长/监护人同意

我已阅读标准版知情同意书，理解研究内容，同意我的孩子参加本研究。

家长/监护人签名: ________________  日期: ____________
与参与者关系: ________________
```

## 电子知情同意(eConsent)补充要求

> 适用于远程知情同意、电子签名、多媒体知情同意等场景

| 要求 | 说明 |
|------|------|
| 身份验证 | 使用政府颁发的ID或生物识别验证参与者身份 |
| 电子签名 | 符合《电子签名法》和FDA 21 CFR Part 11要求 |
| 理解确认 | 每个章节后设置理解确认问题（答对才能继续） |
| 版本控制 | 记录参与者阅读的ICF版本号和阅读时间 |
| 下载保存 | 参与者可随时下载完整的ICF副本 |
| 退出机制 | 提供在线退出通道，退出后确认邮件通知 |
| 审计日志 | 记录所有操作（打开/阅读/签名/退出）的时间戳 |
