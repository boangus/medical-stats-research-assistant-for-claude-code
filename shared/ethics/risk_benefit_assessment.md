# 风险-受益评估框架 (Risk-Benefit Assessment Framework)

> 版本: v1.0.0 | 更新: 2026-06-26
> 适用: 所有涉及人的医学研究
> 参考: 赫尔辛基宣言(2013) §16-17 | CIOMS(2016) 指南 | 《涉及人的生物医学研究伦理审查办法》(2016)

---

## 一、研究风险分类

### 1.1 风险等级定义

| 风险等级 | 定义 | 示例 | IRB审查级别 |
|---------|------|------|------------|
| **最低风险** | 日常生活中的正常风险水平 | 问卷调查、匿名数据分析、常规体检 | 快速审查/豁免 |
| **轻微风险** | 略高于最低风险，不适短暂可逆 | 抽血(少量)、皮肤活检、药物副作用(轻微) | 快速审查 |
| **中度风险** | 可能造成中等程度不适或暂时性伤害 | 影像学检查(辐射)、药物试验(I期后)、侵入性操作 | 全面审查 |
| **高于最小风险** | 可能造成严重但可逆的伤害 | 手术、高风险药物试验、基因治疗 | 全面审查+DSMB |
| **重大风险** | 可能造成不可逆伤害或危及生命 | 首次人体试验、高风险器械、干细胞治疗 | 全面审查+DSMB+期中分析 |

### 1.2 风险评估矩阵

| 风险因素 | 评估问题 | 评分(1-5) |
|---------|---------|----------|
| 身体伤害 | 干预可能造成多大程度的身体伤害？ | |
| 心理伤害 | 研究过程可能造成多大心理压力？ | |
| 隐私泄露 | 个人敏感信息泄露的风险有多大？ | |
| 社会风险 | 参与研究可能带来哪些社会歧视风险？ | |
| 时间负担 | 参与研究需要多少额外时间？ | |
| 经济负担 | 参与研究可能产生哪些额外费用？ | |

**综合风险评分** = 各因素评分之和 / 6
- 1.0-2.0: 最低/轻微风险
- 2.1-3.5: 中度风险
- 3.6-5.0: 高于最小/重大风险

## 二、受益评估框架

### 2.1 受益类型

| 受益类型 | 定义 | 示例 |
|---------|------|------|
| **直接受益** | 参与者本人可能获得的健康改善 | 新药疗效、手术改善症状 |
| **间接受益** | 参与者本人不直接受益，但同类患者受益 | 安慰剂对照组、诊断方法验证 |
| **社会受益** | 医学知识进步，未来患者受益 | 基础研究、流行病学调查 |
| **经济受益** | 参与者获得的经济补偿 | 交通补贴、误工补偿 |

### 2.2 受益评估问题

1. 参与者有多大概率获得直接健康受益？
2. 如不参加本研究，是否有等效的替代治疗方案？
3. 研究结果对同类患者群体的潜在受益有多大？
4. 研究结果对公共卫生政策的潜在影响是什么？

## 三、风险-受益比判定矩阵

### 3.1 判定标准

| 风险等级 | 最低/轻微受益 | 中等受益 | 显著受益 |
|---------|-------------|---------|---------|
| **最低风险** | ✅ 合理 | ✅ 合理 | ✅ 合理 |
| **轻微风险** | ⚠️ 需权衡 | ✅ 合理 | ✅ 合理 |
| **中度风险** | ❌ 不合理 | ⚠️ 需权衡 | ✅ 合理 |
| **高于最小风险** | ❌ 不合理 | ❌ 不合理 | ⚠️ 需权衡 |
| **重大风险** | ❌ 不合理 | ❌ 不合理 | ❌ 不合理(除非无替代) |

### 3.2 判定说明

- **✅ 合理**: 风险-受益比可接受，可批准研究
- **⚠️ 需权衡**: 需要额外的保护措施（如DSMB、期中分析、更严格的入排标准）
- **❌ 不合理**: 风险过大或受益不足，不应批准（除非无替代治疗且参与者知情同意）

## 四、DSMB组建与运作模板

### 4.1 DSMB成员构成

| 角色 | 人数 | 要求 |
|------|------|------|
| 主席 | 1 | 独立统计学家或临床专家，非研究者 |
| 统计学家 | 1-2 | 熟悉期中分析方法和α消耗函数 |
| 临床专家 | 1-2 | 研究疾病领域的资深临床医生 |
| 伦理学家 | 0-1 | 高风险研究必须包含 |

**独立性要求**: DSMB成员不得是研究者、不得持有研究相关利益冲突。

### 4.2 DSMB章程核心内容

```
DSMB章程 — [研究标题]

# 1. 成员名单与利益冲突声明
[列出所有成员及COI声明]

# 2. 会议频率
- 常规会议: 每 [3/6] 个月一次
- 紧急会议: 安全信号出现时48小时内召集

# 3. 期中分析计划
- 分析时点: [N]% 受试者完成主要终点评估时
- α消耗函数: [Lan-DeMets/O'Brien-Fleming/Pocock]
- 独立访问数据频率: 每 [N] 例

# 4. 停止规则
- 安全性停止: 严重不良事件率超过 [X]% → 建议暂停/终止
- 无效性停止: 条件power < [Y]% → 建议终止
- 优效性停止: 期中分析P < [Z] → 建议提前终止

# 5. 保密协议
- 期中数据不得向研究者/申办方泄露
- DSMB会议记录保密保存
- 仅向IRB/伦理委员会报告结论性建议

# 6. 报告模板
- DSMB会议纪要模板
- 安全性报告模板
- 期中分析报告模板
```

## 五、弱势群体研究的风险-受益评估强化

### 5.1 强化评估要求

| 弱势群体 | 额外风险评估 | 额外保护措施 | 中国法规要求 |
|---------|------------|------------|------------|
| **儿童(0-17岁)** | 发育影响、长期安全性 | 儿童版ICF+家长同意+儿童知情同意(≥7岁) | 《未成年人保护法》+伦理审查办法§20 |
| **孕妇** | 胎儿风险、致畸性 | 妊娠检测+避孕要求+胎儿监测 | 伦理审查办法§21 |
| **认知障碍者** | 理解能力评估 | 法定代理人同意+能力恢复后重新同意 | 伦理审查办法§22 |
| **囚犯** | 自愿性评估(胁迫风险) | 独立倡导者+确保不影响假释 | 伦理审查办法§23 |
| **经济困难者** | 经济胁迫评估 | 合理补偿但不过度诱导 | 伦理审查办法§24 |
| **急诊患者** | 紧急状态下的理解能力 | 紧急知情同意+延迟同意 | 伦理审查办法§25 |

### 5.2 强化评估检查清单

- [ ] 是否纳入弱势群体？如是，需说明科学必要性
- [ ] 风险是否为最低风险？如不是，需额外伦理论证
- [ ] 是否有直接健康受益？如没有，需额外保护措施
- [ ] 知情同意过程是否考虑了特殊理解能力？
- [ ] 是否有独立的弱势群体权益保护机制？
- [ ] 补偿是否合理，不会构成不当诱导？
