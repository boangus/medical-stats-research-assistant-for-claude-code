# 变量选择与方法推荐指南

> MSRA 项目专用。基于微信推文和统计学原则，提供变量选择和统计方法推荐。

---

## 1. 基线变量筛选原则

### 1.1 变量分类

| 类型 | 说明 | 纳入原则 |
|------|------|---------|
| **人口学变量** | 年龄、性别、种族、BMI | 强制纳入（无论p值） |
| **临床基线变量** | 疾病严重程度、病程、合并症 | 强制纳入（临床重要性） |
| **暴露/治疗变量** | 主要研究因素 | 强制纳入（研究核心） |
| **潜在混杂因素** | 与暴露和结局均相关 | 强制纳入（DAG指导） |
| **预后因素** | 影响结局的变量 | 建议纳入 |
| **效应修饰因素** | 影响暴露-结局关系的变量 | 亚组分析考虑 |

### 1.2 变量筛选决策树

```
变量是否与研究问题直接相关？
├── 是 → 强制纳入（不看p值）
└── 否 → 检查是否为已知混杂因素
         ├── 是 → 强制纳入（DAG指导）
         └── 否 → 检查是否为预后因素
                  ├── 是 → 建议纳入
                  └── 否 → 单因素分析p<0.1 → 考虑纳入
                           └── p≥0.1 → 不纳入
```

### 1.3 常见基线变量清单

**RCT研究**：
| 变量 | 类型 | 纳入理由 |
|------|------|---------|
| 年龄 | 连续 | 强制纳入（预后因素） |
| 性别 | 分类 | 强制纳入（预后因素） |

## 2. 统计方法选择推荐

### 2.1 方法选择决策树

```
研究设计类型？
├── RCT → 随机化已控制混杂
│   ├── 连续结局 → ANCOVA（调整基线）
│   ├── 二分类结局 → Logistic回归
│   ├── 生存结局 → Cox回归
│   └── 计数结局 → Poisson回归
│
├── 观察性研究 → 需控制混杂
│   ├── 暴露-结局关联 → PSM/IPTW + 多变量调整
│   ├── 预后因素 → Cox回归 + 多变量调整
│   └── 预测模型 → Lasso/RF/XGBoost + 交叉验证
│
└── 诊断试验 → 金标准比较
    ├── 诊断准确性 → ROC分析 + AUC比较
    └── 一致性 → Kappa/ICC + Bland-Altman
```

### 2.2 各方法适用条件

| 方法 | 适用条件 | 假设检验 | 替代方案 |
|------|---------|---------|---------|
| **ANCOVA** | RCT连续结局 | 正态性+方差齐性+回归斜率齐性 | 非参数ANCOVA |
| **Logistic回归** | 二分类结局 | 线性logit+无完全分离 | Firth校正 |
| **Cox回归** | 生存结局 | 比例风险假设 | 分层Cox/时依系数 |
| **PSM** | 观察性研究 | 平衡性（SMD<0.1） | IPTW/OW |
| **Lasso** | 高维数据 | 稳定性选择 | Ridge/Elastic Net |
| **ROC分析** | 诊断试验 | 金标准可靠 | 偏ROC/决策曲线 |

### 2.3 效应量选择

| 研究类型 | 主要效应量 | 计算方式 | 临床解读 |
|---------|-----------|---------|---------|
| RCT（连续） | 均数差（MD） | 组间均数差 | 绝对差异 |
| RCT（二分类） | OR/RR | Logistic/Log-binomial | 相对风险 |
| 生存分析 | HR | Cox回归系数指数化 | 风险比 |
| 诊断试验 | AUC | ROC曲线下面积 | 区分能力 |
| 相关性 | r | Pearson/Spearman | 关联强度 |

## 3. 变量构建推荐与公式

### 3.1 常用变量构建公式

**人口学变量**：
| 变量 | 公式 | 依据 | 缺失处理 |
|------|------|------|---------|
| 年龄 | 入组日期 - 出生日期（年） | 标准人口学 | 删除 |
| BMI | 体重(kg) / (身高(m))² | WHO标准 | 标记缺失 |
| 年龄分组 | <60 → young; 60-70 → middle; ≥70 → old | 临床惯例 | 归入missing组 |

**疾病严重程度**：
| 变量 | 公式 | 依据 | 缺失处理 |
|------|------|------|---------|
| Charlson合并症指数 | 标准计算公式 | Charlson 1987 | 归入0分 |
| ECOG评分 | 标准评分 | ECOG标准 | 标记缺失 |
| Glasgow昏迷评分 | 标准计算 | Glasgow标准 | 标记缺失 |

**实验室指标**：
| 变量 | 公式 | 依据 | 缺失处理 |
|------|------|------|---------|
| eGFR | CKD-EPI公式 | KDIGO指南 | 标记缺失 |
| HbA1c | 直接测量值 | 标准实验室 | 标记缺失 |
| 同型半胱氨酸 | 直接测量值 | 标准实验室 | 标记缺失 |

**结局变量**：
| 变量 | 公式 | 依据 | 缺失处理 |
|------|------|------|---------|
| 治疗反应 | RECIST v1.1: CR+PR vs SD+PD | 实体瘤疗效标准 | 无评估→non-responder |
| 疾病进展 | 基线→终点变化值 | 研究定义 | 多重插补 |
| 复发时间 | 首次复发日期 - 入组日期 | 生存分析标准 | 删失处理 |

### 3.2 变量构建检查清单

```
□ 变量名是否清晰且唯一？
□ 变量类型是否正确（连续/分类/有序）？
□ 构造公式是否可复现？
□ 切点/分类是否有文献依据？
□ 缺失处理策略是否明确？
□ 是否需要处理极端值？
□ 是否需要标准化/归一化？
□ 与SAP定义是否一致？
```
