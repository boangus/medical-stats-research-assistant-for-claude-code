# 通俗术语表 (Plain Language Glossary)

> 版本: v1.0.0 | 更新: 2026-06-26
> 用途: 将统计学术语翻译为日常语言，帮助非统计专业人员理解工具输出

---

## 研究设计术语

| 统计术语 | 通俗解释 | 举例 |
|---------|---------|------|
| **RCT (随机对照试验)** | 把参与者随机分成两组，一组用新药，一组用旧药或安慰剂，看哪个效果更好 | "就像抛硬币决定谁先接受新治疗" |
| **队列研究** | 跟踪一群人很长时间，观察哪些人得了病，哪些人没有，找规律 | "就像跟踪1000个人10年，看谁吸烟谁不吸烟，最后谁得了肺癌" |
| **病例对照研究** | 先找到得病的人和没得病的人，然后回头看他们以前的生活有什么不同 | "先找到肺癌患者和健康人，然后问他们以前吸不吸烟" |
| **诊断试验** | 检查一种新的检测方法(如验血)能不能准确判断谁有病谁没病 | "看看这个新的血液检测能不能准确查出糖尿病" |
| **系统综述** | 把所有关于同一个问题的研究放在一起，综合分析得出更可靠的结论 | "把全世界50个关于降压药的研究放一起看，得出一个更靠谱的结论" |
| **Meta分析** | 用统计方法把多个研究的结果合并成一个总结果 | "把50个研究的数据合在一起算，样本量更大，结论更可靠" |

## 统计分析术语

| 统计术语 | 通俗解释 | 举例 |
|---------|---------|------|
| **P值** | 表示"如果治疗完全没效果，看到当前结果的可能性有多大"。P值越小，说明治疗越可能真的有效 | "P=0.03意味着如果新药完全没用，只有3%的机会看到这么好的效果" |
| **置信区间(CI)** | 真实效果可能在哪个范围内。95%CI意味着我们有95%的把握真实效果在这个范围内 | "HR=0.6, 95%CI(0.4-0.8)意味着新药可能降低20%-60%的风险" |
| **效应量** | 治疗效果有多大，用数字表示 | "Cohen's d=0.8表示效果很大，0.2表示效果很小" |
| **风险比(HR)** | 用新药的人发生某事件的风险是用旧药的人的多少倍。HR<1表示新药更好 | "HR=0.6意味着用新药的人风险降低了40%" |
| **优势比(OR)** | 类似风险比，但用于已经发生的事情。OR<1表示新药更好 | "OR=0.5意味着用新药的人得病的几率只有一半" |
| **显著性** | P值小于0.05时，我们说结果"有统计学意义"，意思是不太可能是偶然 | "P=0.02<0.05，所以结果是显著的，新药很可能真的有效" |
| **多重比较** | 同时检验很多个问题时，需要更严格的标准，否则容易误判 | "同时检验10种药，需要用更严格的标准，否则可能把没效的药说成有效" |

## 数据质量术语

| 统计术语 | 通俗解释 | 举例 |
|---------|---------|------|
| **缺失值** | 某些参与者没有提供某些信息 | "100个人中有5个人没填体重，这5个体重就是缺失值" |
| **异常值** | 和其他人差别很大的数据，可能是测量错误，也可能是真实情况 | "99个人身高在150-190cm，有1个人是250cm，这个就是异常值" |
| **正态分布** | 数据分布像钟形曲线，中间多两边少，大多数人的值在平均值附近 | "大部分人的身高在平均值附近，特别高和特别矮的人都很少" |
| **偏态分布** | 数据分布不对称，一边的尾巴比另一边长 | "收入分布就是偏态的，少数人收入特别高，把平均值拉高了" |
| **去标识化** | 把能识别出个人身份的信息(姓名、身份证号等)去掉或替换 | "把张三的名字换成'参与者001'，这样就认不出是谁了" |

## 研究质量术语

| 统计术语 | 通俗解释 | 举例 |
|---------|---------|------|
| **SAP (统计分析计划)** | 在看数据之前就定好怎么分析，防止事后根据结果调整分析方法 | "就像考试前先定好评分标准，不能考完再改" |
| **ITT (意向治疗分析)** | 所有被随机分组的人都算在内，不管他们有没有完成治疗 | "即使有人中途退出，也要按最初分的组来算" |
| **PP (符合方案分析)** | 只算那些完全按照方案完成治疗的人 | "只算那些按时吃完了所有药的人" |
| **混杂因素** | 除了研究的治疗外，其他可能影响结果的因素 | "吸烟的人可能同时喝酒多，所以喝酒就是混杂因素" |
| **偏倚** | 研究设计或执行中的系统性误差，导致结果偏离真实值 | "只调查住院病人会高估疾病严重程度，这就是选择偏倚" |
| **质量门闸** | 在分析过程中设置的检查点，确保数据和方法质量达标 | "就像工厂的质量检查，不合格的产品不能出厂" |

## 结果解释术语

| 统计术语 | 通俗解释 | 举例 |
|---------|---------|------|
| **临床意义** | 治疗效果是否大到对患者有实际价值(不只是统计上显著) | "降压药统计上显著但只降了1mmHg，对患者没什么实际意义" |
| **敏感性分析** | 换一种方法重新分析，看结论是否一致 | "用不同的统计方法再算一遍，如果结论一样就更可信" |
| **亚组分析** | 在特定人群中看治疗效果是否有差异 | "看看新药对老年人和年轻人的效果是否一样" |
| **E值** | 表示需要多强的未测量混杂因素才能推翻当前结论。E值越大，结论越可靠 | "E=2.7意味着需要一个比已知风险因素都强的混杂因素才能推翻结论" |
| **竞争风险** | 有多种可能的结局，一种结局的发生可能阻止另一种结局的发生 | "心血管死亡和非心血管死亡是竞争风险——死于癌症的人不会死于心脏病" |
