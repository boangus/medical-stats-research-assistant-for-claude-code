# 速查表 (Cheatsheet)

## 研究设计决策表

| 研究目的 | 推荐设计 | 关键考虑 |
|---------|---------|---------|
| 证明新疗法更有效 | 优效性试验 | 样本量、终点选择 |
| 证明新疗法不劣于标准治疗 | 非劣效性试验 | 界值选择、样本量更大 |
| 评估真实世界效果 | 实效性试验 | 外部效度、代表性 |
| 群组层面干预 | 整群随机试验 | ICC、样本量调整 |
| 质量改进干预 | 阶梯楔形试验 | 时间趋势、群组数量 |

## 样本量计算要素

| 要素 | 说明 | 常用值 |
|-----|------|-------|
| α (一类错误) | 假阳性概率 | 0.05 (双侧) |
| 1-β (检验效能) | 发现真实差异的概率 | 0.80 或 0.90 |
| 效应量 | 预期差异大小 | 基于临床意义 |
| MCID | 最小临床重要差异 | 患者视角 |

## 统计方法选择

| 数据类型/问题 | 推荐方法 | 章节 |
|--------------|---------|-----|
| 二分类结局 | Logistic回归 | 第15章 |
| 时间至事件 | 生存分析/Cox回归 | 第10章 |
| 重复测量 | 混合模型 | 第14章 |
| 多个研究合并 | Meta分析 | 第26章、第37章 |
| 缺失数据 | 多重插补 | 第20章 |
| 多重比较 | Bonferroni/FDR | 第18章、第19章 |
| 混杂控制 | 倾向性评分 | 第35章 |

## 效应量解读

| 指标 | 解读 |
|-----|-----|
| OR > 1 | 风险增加 |
| OR < 1 | 风险降低 |
| HR > 1 | 风险增加 |
| HR < 1 | 风险降低 |
| NNT = 1/RD | 预防一例不良事件需治疗人数 |

## 偏倚类型速查

| 偏倚类型 | 防范措施 |
|---------|---------|
| 选择偏倚 | 随机化、明确纳入标准 |
| 信息偏倚 | 标准化测量、盲法 |
| 混杂偏倚 | 多变量分析、倾向性评分 |
| 指示性混杂 | 工具变量、倾向性评分 |

## 数据库研究检查清单

1. 明确研究问题和假说
2. 选择合适数据库
3. 明确变量定义
4. 识别混杂因素
5. 处理缺失数据
6. 进行敏感性分析
7. 避免因果推断语言
