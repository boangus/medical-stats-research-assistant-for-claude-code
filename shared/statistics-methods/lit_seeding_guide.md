# 文献种子检索策略指南（Lit-Seeding Guide）

> 用于 analysis-plan Phase 1.5，为统计方法选择提供实证依据。

---

## 1. 目的

文献种子检索**不是**完整文献综述，而是快速定位 3-5 篇核心参考文献，用于：
- 了解同类研究的统计方法选择
- 获取效应量估计（用于样本量计算）
- 确认终点定义的行业惯例
- 识别方法学争议点

## 2. 检索策略

### 2.1 关键词提取

从研究问题中提取 PICO/PECO 格式关键词：

| 要素 | 提取内容 | 示例 |
|------|---------|------|
| P (Population) | 目标人群特征 | "2型糖尿病成人患者" |
| I/E (Intervention/Exposure) | 干预或暴露 | "GLP-1受体激动剂" |
| C (Comparator) | 对照 | "安慰剂 / 标准治疗" |
| O (Outcome) | 结局指标 | "HbA1c变化 / 心血管事件" |

### 2.2 检索执行

使用 `deep-research` skill 的 `quick` 模式：
```
/deep-research --quick "[PICO关键词] 统计方法 样本量 效应量"
```

**检索优先级**：
1. 同研究类型的 Meta-analysis（效应量汇总最权威）
2. 同研究类型的大型 RCT（方法选择最规范）
3. 近 3 年的系统综述（方法学趋势）
4. 临床指南中的统计方法推荐

### 2.3 信息提取

对每篇文献提取以下信息：

| 字段 | 说明 | 用途 |
|------|------|------|
| 研究设计 | RCT / 队列 / 病例对照 | 确认方法体系 |
| 主要终点 | 具体定义 + 测量方式 | 参考终点定义 |
| 分析方法 | 主分析 + 敏感性分析 | 方法选择参考 |
| 样本量 | N + 计算依据 | 样本量参考 |
| 效应量 | d / OR / HR + 95% CI | 效应量估计 |
| 调整变量 | 协变量列表 | 协变量选择参考 |

## 3. 输出格式

生成 `literature_seeds.md`：

```markdown
# 文献种子（Literature Seeds）

> 来源：Phase 1.5 Lit-Seeding（非正式文献综述，Stage 5.1 将重新验证）

| # | 文献 | 研究设计 | 主要终点 | 分析方法 | 样本量 | 效应量 |
|---|------|---------|---------|---------|--------|--------|
| 1 | Smith 2023 | RCT | HbA1c变化 | ANCOVA | 320 | d=0.45 |
| 2 | Chen 2022 | RCT | HbA1c变化 | MMRM | 450 | d=0.38 |
| 3 | Meta-X 2024 | SR/MA | 心血管事件 | 网络Meta | 12,000 | HR=0.82 |

## 方法选择参考
- 主分析：ANCOVA（文献 1,2 均使用，调整基线）
- 样本量：基于 d=0.42（文献均值），α=0.05，β=0.2 → N=180/组
- 敏感性：MMRM（处理缺失数据）
```

## 4. 注意事项

### ⚠️ Anti-patterns

| 禁止行为 | 正确做法 |
|---------|---------|
| 将 lit-seeding 结果直接复制到论文参考文献 | Stage 5.1 正式文献调研时重新验证并引用 |
| 仅引用 1 篇文献确定效应量 | 取多篇文献的均值或中位数 |
| 忽略阴性结果文献 | 平衡引用阳性/阴性结果 |
| 使用 >5 年前的文献确定方法 | 优先引用近 3 年文献 |

### 降级策略

- `deep-research` 不可用 → 提示用户手动提供 2-3 篇参考文献
- 用户已知文献 → 直接纳入文献种子，标注"用户提供"
- 检索无结果 → 标注 `[SKIP: 未找到相关文献]`，继续 Phase 2

## 5. 融合点 C（文献 seed）增强

| 阶段 | 原流程 | 优化后 |
|------|--------|--------|
| MSRA Phase 1 | 无文献检索 | Phase 1.5 literature_seeds |
| Stage 5.1 | deep-research 从零开始 | 基于 literature_seeds 精准扩展 |
