# 方法学模式库 (Patterns)

## 研究设计模式

### 1. 非劣效性试验设计
**何时使用**：新疗法预期与标准治疗疗效相似，但具有其他优势。
**如何实施**：确定非劣效性界值 → 选择有效对照组 → 计算样本量 → 单侧置信区间分析 → ITT+符合方案集分析。
**权衡**：样本量较大；界值选择主观性。

### 2. 整群随机试验设计
**何时使用**：干预需在群组层面实施，或存在污染风险。
**如何实施**：确定随机单位 → 估计ICC → 调整样本量 → 广义估计方程分析。
**权衡**：需要更大样本量；群组间可能不平衡。

### 3. 阶梯楔形试验设计
**何时使用**：评估质量改进干预，无法同时实施。
**如何实施**：确定群组和步骤数量 → 随机分配干预启动时间 → 混合模型分析。
**权衡**：分析复杂；可能受时间趋势影响。

## 分析方法模式

### 4. 缺失数据处理
**何时使用**：研究数据存在缺失。
**如何实施**：判断缺失机制(MCAR/MAR/MNAR) → 选择方法 → 敏感性分析。
**权衡**：方法选择影响结果；MNAR难以处理。

### 5. 倾向性评分方法
**何时使用**：观察性研究需控制大量混杂因素。
**如何实施**：估计倾向性评分 → 选择方法(匹配/分层/加权) → 评估平衡性 → 敏感性分析。
**权衡**：只能控制已测量混杂。

### 6. 多重比较校正
**何时使用**：多个终点或多次比较。
**如何实施**：确定比较次数 → 选择校正方法(Bonferroni/Holm/FDR) → 报告校正后P值。
**权衡**：降低假阳性但增加假阴性。

## 结果报告模式

### 7. 效应量报告
**何时使用**：报告研究结果。
**如何实施**：报告点估计和置信区间 → 计算临床意义指标(NNT/RD) → 结合临床背景解读。

### 8. 风险预测模型评估
**何时使用**：开发或验证预测模型。
**如何实施**：评估区分度(C统计量) → 评估校准度 → 内部验证 → 外部验证。
**权衡**：区分度和校准度可能不一致。
