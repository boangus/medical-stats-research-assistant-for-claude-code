## 4. 研究样本

### 4.1 样本量计算

#### 4.1.1 基于敏感度/特异度的精度法（Buderer公式）

> 适用于按预设敏感度/特异度估计所需样本量。

**公式（Buderer 1996）**：

```
n_disease = (Z_{1-α/2}² × Se × (1-Se)) / (d × p_disease)²
n_no_disease = (Z_{1-α/2}² × Sp × (1-Sp)) / (d × (1-p_disease))²

其中：
  Se = 预期敏感度
  Sp = 预期特异度
  d = 允许的精度（半宽，如 0.05）
  p_disease = 目标人群患病率
  Z_{1-α/2} = 标准正态分位数（α=0.05 时为 1.96）
  n_disease = 所需患病人数
  n_no_disease = 所需非患病人数
  N = n_disease + n_no_disease
```

**参数设定**：

| 参数 | 设定值 | 依据 |
|------|--------|------|
| 预期敏感度（Se） | [{如：0.85}] | [{文献依据/预试验}] |
| 预期特异度（Sp） | [{如：0.90}] | [{文献依据/预试验}] |
| 允许精度（d） | [{如：0.05}] | [{临床可接受精度}] |
| 患病率（p） | [{如：0.30}] | [{目标人群估计}] |
| α | 0.05（双侧） | 常规设定 |
| 计算结果 N | [{计算样本量}] | — |
| 脱落率调整 | [{如：10%}] | [{依据}] |
| **最终样本量** | **[{N_final}]** | — |

#### 4.1.2 基于AUC的Obuchowski公式

> 适用于以AUC为主要指标的样本量估计。

**公式（Obuchowski 1997）**：

```
n = [(Z_{1-α/2} × sqrt(V0(AUC)) + Z_{1-β} × sqrt(V1(AUC))) / (AUC1 - AUC0)]²

其中：
  AUC0 = 零假设下的AUC（通常为 0.5）
  AUC1 = 预期AUC
  V0(AUC) = (0.0099) × [e^{-0.5×a²} × (5×a² + 8) + a² + 4]  （a = Φ^{-1}(AUC0)）
  V1(AUC) = (0.0099) × [e^{-0.5×b²} × (5×b² + 8) + b² + 4]  （b = Φ^{-1}(AUC1)）
  Φ^{-1} = 标准正态分布的反函数
```

**参数设定**：

| 参数 | 设定值 | 依据 |
|------|--------|------|
| 预期AUC（AUC1） | [{如：0.85}] | [{文献依据}] |
| 零假设AUC（AUC0） | 0.5 | 无判别力 |
| α | 0.05（双侧） | 常规设定 |
| Power（1-β） | [{如：0.80}] | 常规设定 |
| 计算结果 N | [{计算样本量}] | — |
| **最终样本量** | **[{N_final}]** | — |

## 6. 统计方法

### 6.1 描述性统计

- **连续变量**：均值（标准差）或中位数 [IQR]，按金标准分组
- **分类变量**：频数（百分比），按金标准分组
- **基线均衡性**：SMD < 0.1 为均衡（注意：诊断研究中基线差异为疾病谱特征，非随机化所致）

### 6.2 2×2 列联表

```
                    金标准

待评价试验  阳性    TP          FP          TP+FP
            阴性    FN          TN          FN+TN
            合计    TP+FN       FP+TN       N
```

| 指标 | 计算公式 | 95% CI 方法 |

| 分析内容 | 方法 | 输出 |

### 6.9 决策曲线分析（DCA）

> 评估不同阈值概率下待评价试验的临床净获益。

- **方法**：计算一系列阈值概率（0~1）下的净获益
- **净获益公式**：$NB = TP/N - FP/N \times [p_t/(1-p_t)]$（$p_t$ = 阈值概率）
- **比较策略**：[{待评价试验 vs 全部治疗 vs 全部不治疗 vs 其他策略}]
- **报告**：DCA曲线图，标注临床合理阈值范围

### A.2 Obuchowski AUC样本量公式

```python
# Python代码示例
from scipy.stats import norm
import numpy as np

def obuchowski_n(auc1, auc0=0.5, alpha=0.05, power=0.80):
    z_alpha = norm.ppf(1 - alpha/2)
    z_beta = norm.ppf(power)
    a = norm.ppf(auc0)
    b = norm.ppf(auc1)
    V0 = 0.0099 * (np.exp(-0.5*a**2) * (5*a**2 + 8) + a**2 + 4)
    V1 = 0.0099 * (np.exp(-0.5*b**2) * (5*b**2 + 8) + b**2 + 4)
    n = ((z_alpha * np.sqrt(V0) + z_beta * np.sqrt(V1)) / (auc1 - auc0))**2
    return int(np.ceil(n))
```

### A.3 似然比与DOR

```
LR+ = Se / (1 - Sp)
LR- = (1 - Se) / Sp
DOR = LR+ / LR- = (TP × TN) / (FP × FN)

95% CI for DOR:
  ln(DOR) ± 1.96 × SE(ln(DOR))
  SE(ln(DOR)) = sqrt(1/TP + 1/FP + 1/FN + 1/TN)
```

### A.4 Fagan图验后概率

```
验后概率 = (LR × 验前比) / (1 + LR × 验前比)
验前比 = 验前概率 / (1 - 验前概率)
```
