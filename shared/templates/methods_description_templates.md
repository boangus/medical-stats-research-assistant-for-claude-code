## 1. 描述性统计模板

### 1.1 基线特征描述

**模板1: 连续变量**
```
基线连续变量以均数±标准差（正态分布）或中位数（四分位距）（偏态分布）描述。
组间比较采用独立样本t检验（正态分布）或Mann-Whitney U检验（偏态分布）。
```

**模板2: 分类变量**
```
基线分类变量以频数（百分比）描述。
组间比较采用χ²检验（预期频数≥5）或Fisher精确检验（预期频数<5）。
```

**模板3: 综合描述**
```
表1展示了研究人群的基线特征。连续变量以均数±标准差（正态分布）或中位数（四分位距）（偏态分布）描述，分类变量以频数（百分比）描述。
组间比较：连续变量采用独立样本t检验（正态分布）或Mann-Whitney U检验（偏态分布）；分类变量采用χ²检验（预期频数≥5）或Fisher精确检验（预期频数<5）。
```

## 附录: 常用统计方法描述

### A. 参数检验

**独立样本t检验**
```
采用独立样本t检验比较两组连续变量的差异。
正态性检验通过Shapiro-Wilk检验（p>0.05）和Q-Q图评估。
方差齐性通过Levene检验评估（p>0.05为方差齐性）。
```

**单因素方差分析**
```
采用单因素方差分析（ANOVA）比较多组连续变量的差异。
正态性检验通过Shapiro-Wilk检验（p>0.05）和Q-Q图评估。
方差齐性通过Levene检验评估（p>0.05为方差齐性）。
如存在统计学显著差异，进一步采用Tukey HSD进行事后两两比较。
```

### B. 非参数检验

**Mann-Whitney U检验**
```
采用Mann-Whitney U检验比较两组连续变量的差异（数据不满足正态分布假设）。
效应量以中位数差及其95%置信区间表示。
```

**Kruskal-Wallis检验**
```
采用Kruskal-Wallis检验比较多组连续变量的差异（数据不满足正态分布假设）。
如存在统计学显著差异，进一步采用Dunn检验进行事后两两比较（Bonferroni校正）。
```

### C. 回归分析

**线性回归**
```
采用线性回归评估[暴露]与[结局]的关联。
模型拟合优度通过R²评估。
残差分析通过残差图、Q-Q图和Shapiro-Wilk检验评估模型假设。
```

**Logistic回归**
```
采用Logistic回归评估[暴露]与[二分类结局]的关联。
效应量以优势比（OR）及其95%置信区间表示。
模型校准通过Hosmer-Lemeshow检验评估。
```

**Cox回归**
```
采用Cox比例风险回归评估[暴露]与[生存结局]的关联。
效应量以风险比（HR）及其95%置信区间表示。
比例风险假设通过Schoenfeld残差检验评估。
```
