## 1. 评估目的

### 1.1 评估背景

多中心临床试验中，不同中心可能因患者人群特征、医疗实践、操作差异等因素导致结局或处理效应存在异质性。中心效应评估旨在：

1. **量化中心间变异**：评估各中心在基线特征、结局指标上的差异程度
2. **评估处理效应一致性**：检验处理效应是否在各中心间一致（交互效应）
3. **指导分析策略**：根据异质性程度选择合适的统计模型（固定效应/随机效应/分层分析）
4. **满足监管要求**：ICH E9 第5.4节要求多中心试验需评估中心效应

### 1.2 评估范围

| 评估维度 | 评估内容 | 主要/探索性 |
|---------|---------|-----------|
| 中心间基线差异 | 各中心人口学、疾病特征比较 | 描述性 |
| 中心间结局差异 | 各中心主要/次要结局比较 | 描述性 |
| 处理×中心交互 | 处理效应是否随中心变化 | 主要 |
| 中心间异质性 | I²、τ²、Cochran's Q | 主要 |
| 中心效应调整 | 固定/随机效应模型比较 | 探索性 |

### 2.2 模型方法

#### 2.2.1 固定效应模型（中心作为协变量）

> 将中心作为固定效应纳入模型，调整中心间差异。

```
模型1（不含交互项）：
  y_ij = β₀ + δ × treat_ij + Σ β_k × x_kij + γ_j × center_j + ε_ij

模型2（含交互项）：
  y_ij = β₀ + δ × treat_ij + Σ β_k × x_kij + γ_j × center_j + ψ_j × (treat_ij × center_j) + ε_ij

其中：
  δ = 汇总处理效应（主要关注参数）
  γ_j = 中心j的固定效应
  ψ_j = 中心j的处理效应修饰（交互项）
```

```r
# R代码示例（连续结局）
# 模型1：固定效应，无交互
model1 <- lm(y ~ treat + center + covariate1 + covariate2, data = df)

# 模型2：固定效应，含交互
model2 <- lm(y ~ treat * center + covariate1 + covariate2, data = df)

# 交互检验
anova(model1, model2)  # 比较两模型
```

#### 2.2.2 随机效应模型（中心随机截距/斜率）

> 将中心作为随机效应，允许中心间变异，适用于中心较多时。

```
随机截距模型：
  y_ij = β₀ + δ × treat_ij + Σ β_k × x_kij + u_0j + ε_ij
  u_0j ~ Normal(0, τ²_center)

随机截距+随机斜率模型：
  y_ij = β₀ + δ × treat_ij + Σ β_k × x_kij + u_0j + u_1j × treat_ij + ε_ij
  (u_0j, u_1j) ~ Bivariate Normal(0, Σ)
```

```r
# R代码示例（lme4包）
library(lme4)

# 随机截距模型
model_ri <- lmer(y ~ treat + covariate1 + covariate2 + (1 | center), data = df)

# 随机截距+随机斜率模型
model_rs <- lmer(y ~ treat + covariate1 + covariate2 + (1 + treat | center), data = df)

# 模型比较（检验随机斜率是否必要）
anova(model_ri, model_rs)
```

#### 2.2.3 广义估计方程（GEE）

> GEE方法适用于中心作为聚类单元的分析，对中心相关性的建模更为稳健。

```r
# R代码示例（geepack包）
library(geepack)
model_gee <- geeglm(y ~ treat + covariate1 + covariate2,
                    data = df, id = center,
                    corstr = "exchangeable",  # 中心内等相关结构
                    family = gaussian)
summary(model_gee)
```

### 2.3 交互检验（处理×中心交互）

> 检验处理效应是否在各中心间一致，是中心效应评估的核心。

| 检验方法 | 模型 | 检验统计量 | H0 | 判定标准 |
|---------|------|-----------|-----|---------|
| **似然比检验（LRT）** | 模型1（无交互） vs 模型2（含交互） | χ² | 无交互效应 | p < 0.10 提示显著交互 |
| **Wald检验** | 交互项系数联合检验 | χ² | 所有交互项 = 0 | p < 0.10 提示显著交互 |
| **F检验** | 嵌套模型比较（连续结局） | F | 无交互效应 | p < 0.10 提示显著交互 |

**注意**：
- 交互检验使用 α = 0.10（而非0.05），因为交互检验把握度通常较低
- 若中心数量多而每中心样本量少，交互检验可能欠把握，需结合描述性分析

### 2.4 I² 统计量

> I² 统计量量化中心间异质性比例（非抽样误差导致的变异占比）。

#### 2.4.1 计算方法

```
I² = max(0, (Q - df) / Q) × 100%

其中：
  Q = Cochran's Q 统计量（各中心效应量与合并效应量离差平方和）
  df = 中心数 - 1
```

#### 2.4.2 异质性分级

| I² 范围 | 异质性程度 | 处理策略（参考 sap_standard.md §8.5） |
|---------|-----------|--------------------------------------|
| **< 25%** | 低异质性 | 固定效应模型（中心作为协变量） |
| **25% - 50%** | 中等异质性 | 固定效应或随机效应模型 |
| **50% - 75%** | 中高异质性 | 随机效应模型（中心随机截距） |
| **> 75%** | 高异质性 | 随机效应模型 + 报告各中心效应量；考虑两步法meta分析 |

#### 2.4.3 其他异质性指标

| 指标 | 公式 | 解释 |
|------|------|------|
| **τ²（Tau²）** | DerSimonian-Laird估计 | 中心间方差（绝对量），τ² > 0 提示异质 |
| **Cochran's Q** | $\sum w_i (\hat{\theta}_i - \hat{\theta})^2$ | Q检验，p < 0.10 提示显著异质 |
| **H 统计量** | $H = \sqrt{Q / df}$ | H > 1 提示异质，H < 1.2 为低异质 |

```r
# R代码示例（metafor包计算I²）
library(metafor)
# 两步法：先计算各中心效应量
center_effects <- escalc(measure = "MD", m1i = mean_treat, sd1i = sd_treat,
                         n1i = n_treat, m2i = mean_ctrl, sd2i = sd_ctrl,
                         n2i = n_ctrl, data = center_data)
# 固定效应meta分析
res <- rma(yi, vi, data = center_effects, method = "FE")
cat("I² =", res$I2, "%\n")
cat("τ² =", res$tau2, "\n")
cat("Q =", res$QE, ", p =", res$QEp, "\n")
```

## 3. 评估标准

### 3.1 中心效应显著判定标准

> 综合多项指标判定中心效应是否显著。

| 评估指标 | 判定标准 | 显著性 |
|---------|---------|--------|
| **处理×中心交互** | LRT/Wald p < 0.10 | 显著交互 → 处理效应不一致 |
| **I² 统计量** | I² > 50% | 中高异质性 |
| **Cochran's Q** | p < 0.10 | 显著异质 |
| **中心间方差 τ²** | τ² > 0 且95%CI不含0 | 中心间变异显著 |
| **各中心效应方向** | 存在方向不一致的中心 | 定性交互 |
| **leave-one-out** | 排除某中心后结论改变 | 结果不稳定 |

### 3.2 中心效应分级

| 等级 | 标准 | 处理要求 |
|------|------|---------|
| **无显著中心效应** | 交互p ≥ 0.10 且 I² < 25% | 不调整或固定效应调整 |
| **轻度中心效应** | 交互p ≥ 0.10 但 I² 25-50% | 固定效应调整 |
| **中度中心效应** | 交互p < 0.10 或 I² 50-75% | 随机效应调整 |
| **重度中心效应** | 交互p < 0.10 且 I² > 75%，或定性交互 | 分层分析 + 报告各中心效应 |

## 4. 处理策略

> 参考 sap_standard.md §8.5 中心效应处理策略

### 4.1 策略选择决策树

```
                    中心效应评估结果
                         │
              ┌──────────┼──────────┐
              │          │          │
         I² < 25%   I² 25-75%   I² > 75%
         交互p≥0.10  或交互p<0.10  或定性交互
              │          │          │
              ▼          ▼          ▼
          不调整或     随机效应     分层分析
          固定效应     调整        +各中心报告
          调整
```

### 4.2 策略1：不调整

| 项目 | 内容 |
|------|------|
| **适用条件** | I² < 25%，交互p ≥ 0.10，中心间基线均衡 |
| **模型** | 不纳入中心项：$y = β₀ + δ × treat + \text{covariates} + ε$ |
| **输出** | 汇总处理效应及95% CI |
| **优点** | 简单，把握度最高 |
| **风险** | 未控制中心间变异 |
| **敏感性分析** | 必须同时报告固定效应调整结果 |

### 4.3 策略2：固定效应调整

| 项目 | 内容 |
|------|------|
| **适用条件** | I² < 25%（中心间同质），但需控制中心差异 |
| **模型** | 中心作为固定协变量：$y = β₀ + δ × treat + γ_j × center + \text{covariates} + ε$ |
| **输出** | 中心调整后的汇总处理效应 + 中心效应估计 |
| **优点** | 控制中心间系统差异，解释直观 |
| **风险** | 中心数多时参数过多；无法外推到未纳入中心 |
| **R代码** | `lm(y ~ treat + factor(center) + covariates, data=df)` |

### 4.4 策略3：随机效应调整

| 项目 | 内容 |
|------|------|
| **适用条件** | I² 25-75%（中心间异质），中心数 ≥ 6 |
| **模型** | 中心随机截距：$y = β₀ + δ × treat + u_{0j} + \text{covariates} + ε$，$u_{0j} \sim N(0, τ²)$ |
| **输出** | 汇总处理效应 + 中心间方差 τ² |
| **优点** | 可外推到未纳入中心；参数更少；处理中心间变异 |
| **风险** | τ²估计不稳定（中心数少时）；假设正态分布 |
| **R代码** | `lmer(y ~ treat + covariates + (1|center), data=df)` |
| **扩展** | 如需评估处理效应异质性：`(1+treat|center)` |

### 4.5 策略4：分层分析

| 项目 | 内容 |
|------|------|
| **适用条件** | I² > 75% 或定性交互（效应方向不一致） |
| **方法** | 两步法：①各中心内独立估计效应量 → ②随机效应meta汇总 |
| **输出** | 汇总效应量 + I²/τ²/Q + 各中心单独效应量森林图 |
| **优点** | 最灵活，不假设中心间同质；可识别异常中心 |
| **风险** | 小中心估计不稳定；把握度降低 |
| **R代码** | `metafor::rma(yi, vi, method="REML")` |

### 4.6 敏感性分析

| 分析 | 方法 | 目的 |
|------|------|------|
| **Leave-one-out** | 逐一排除各中心，重新估计汇总效应 | 评估结论稳健性，识别影响性中心 |
| **大中心 vs 小中心** | 按样本量中位数分层比较 | 评估中心规模影响 |
| **不同模型比较** | 固定效应 vs 随机效应 vs GEE | 评估模型选择影响 |
| **极端中心排除** | 排除效应量最极端的1-2个中心 | 评估极端值影响 |

```r
# ============================================
# 中心效应评估完整流程
# ============================================

library(lme4)
library(metafor)
library(ggplot2)
library(dplyr)

# --- 1. 描述性分析 ---
# 各中心基线特征
baseline_by_center <- df %>%
  group_by(center) %>%
  summarise(
    N = n(),
    age_mean = mean(age, na.rm = TRUE),
    age_sd = sd(age, na.rm = TRUE),
    male_pct = mean(sex == 1) * 100
  )

# 各中心效应量（两步法）
center_effects <- df %>%
  group_by(center) %>%
  summarise(
    n_treat = sum(treat == 1),
    n_ctrl = sum(treat == 0),
    mean_treat = mean(y[treat == 1], na.rm = TRUE),
    sd_treat = sd(y[treat == 1], na.rm = TRUE),
    mean_ctrl = mean(y[treat == 0], na.rm = TRUE),
    sd_ctrl = sd(y[treat == 0], na.rm = TRUE)
  )

# --- 2. 固定效应模型（无交互） ---
model_fe <- lm(y ~ treat + factor(center) + age + sex, data = df)

# --- 3. 固定效应模型（含交互） ---
model_fe_int <- lm(y ~ treat * factor(center) + age + sex, data = df)

# --- 4. 交互检验（LRT） ---
interaction_test <- anova(model_fe, model_fe_int)
cat("交互检验 p =", interaction_test$`Pr(>F)`[2], "\n")

# --- 5. 随机效应模型 ---
model_re <- lmer(y ~ treat + age + sex + (1 | center), data = df)
model_re_slope <- lmer(y ~ treat + age + sex + (1 + treat | center), data = df)

# --- 6. 异质性指标（两步法meta） ---
escalc_data <- escalc(measure = "MD",
                      m1i = mean_treat, sd1i = sd_treat, n1i = n_treat,
                      m2i = mean_ctrl, sd2i = sd_ctrl, n2i = n_ctrl,
                      data = center_effects)
meta_fe <- rma(yi, vi, data = escalc_data, method = "FE")
meta_re <- rma(yi, vi, data = escalc_data, method = "REML")

cat("I² =", meta_fe$I2, "%\n")
cat("τ² =", meta_re$tau2, "\n")
cat("Q =", meta_fe$QE, ", p =", meta_fe$QEp, "\n")

# --- 7. 森林图 ---
forest(meta_re, slab = paste("中心", escalc_data$center))
```
