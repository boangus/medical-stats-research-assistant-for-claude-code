# CARE 报告规范检查清单

> 适用于病例报告（Case Report）
> 基于CARE指南（2013）

---

## 标题部分

### 1. 标题
- [ ] 标题中包含"病例报告"或"case report"
- [ ] 标题中包含诊断（最重要特征）
- [ ] 标题中包含干预（如适用）

### 2. 关键词
- [ ] 提供2-5个关键词
- [ ] 包含诊断、干预、病例报告

## 摘要部分

### 3. 结构化摘要
- [ ] 背景：介绍病例的独特性
- [ ] 病例介绍：患者 demographics、诊断、干预、结局
- [ ] 结论：主要结论和临床意义

## 引言部分

### 4. 引言
- [ ] 介绍病例的独特性和教学价值
- [ ] 说明报告该病例的理由
- [ ] 提供相关文献简要回顾

## 病例介绍部分

### 5. 患者信息
- [ ] 年龄、性别、种族（如相关）
- [ ] 就诊时间
- [ ] 主诉和现病史
- [ ] 既往史和用药史
- [ ] 家族史和个人史

### 6. 体格检查
- [ ] 生命体征
- [ ] 相关体格检查发现
- [ ] 系统回顾

### 7. 诊断评估
- [ ] 辅助检查结果（实验室、影像、病理等）
- [ ] 诊断过程和鉴别诊断
- [ ] 最终诊断和诊断标准

### 8. 干预措施
- [ ] 治疗方案详细描述
- [ ] 剂量、频率、疗程
- [ ] 治疗时间线
- [ ] 合并用药

### 9. 随访和结局
- [ ] 随访时间和方式
- [ ] 治疗反应和结局
- [ ] 并发症和不良反应
- [ ] 患者满意度（如适用）

### 10. 讨论
- [ ] 病例的独特性和教学价值
- [ ] 与文献比较
- [ ] 病例的局限性
- [ ] 临床启示和建议

## 其他部分

### 11. 患者知情同意
- [ ] 获得患者知情同意
- [ ] 隐私保护措施

### 12. 利益冲突声明
- [ ] 报告所有利益冲突
- [ ] 资金来源

### 13. 参考文献
- [ ] 引用相关文献
- [ ] 遵循期刊引用格式

## 报告要点

### 必须包含的元素
1. **患者 demographics**：年龄、性别、种族
2. **主诉和现病史**：症状、持续时间、进展
3. **体格检查**：关键发现
4. **辅助检查**：实验室、影像、病理结果
5. **诊断**：最终诊断和诊断标准
6. **干预**：详细治疗方案
7. **结局**：治疗反应和随访结果
8. **讨论**：病例的独特性和临床意义

### 建议包含的元素
1. **时间线图**：展示病程进展
2. **患者流程图**：展示治疗过程
3. **影像资料**：关键影像学发现
4. **病理图片**：关键病理学发现

## 常见问题

### Q1: 病例报告需要多少患者？
A: 通常1-3例。单个病例报告需要足够的独特性和教学价值。

### Q2: 需要伦理审批吗？
A: 通常不需要，但需要患者知情同意。

### Q3: 可以报告阴性结果吗？
A: 可以，如果病例有教学价值（如罕见疾病的诊断挑战）。
