# STRICTA 针灸临床研究报告清单

> **STRICTA**: Standards for Reporting Interventions in Clinical Trials of Acupuncture
> 适用于针灸临床试验的干预措施报告
> 参考: MacPherson H, et al. (2010) Revised STandards for Reporting Interventions in Clinical Trials of Acupuncture (STRICTA)
> 更新日期: 2026-06-24

---

## 1. 针灸理论依据 (Acupuncture Rationale)

### Item 1a. 针灸风格/传统
- [ ] 描述针灸的风格或传统（如：中国针灸、日本针灸、韩国针灸、耳针、头皮针等）
- [ ] 如有特定流派或学派，注明来源

### Item 1b. 取穴依据
- [ ] 取穴的理由/依据（文献、共识、专家意见）
- [ ] 引用的相关文献或经典著作
- [ ] 如为个体化取穴，描述选穴的理论依据

### Item 1c. 既往针灸治疗经验
- [ ] 患者是否接受过针灸治疗
- [ ] 如适用，描述既往治疗情况

## 2. 针灸细节 (Details of Needling)

### Item 2a. 穴位选择
- [ ] 主穴名称（使用 WHO 标准穴位命名）
- [ ] 配穴名称（如有）
- [ ] 单侧/双侧取穴

### Item 2b. 进针深度
- [ ] 每个穴位的进针深度（mm 或 cun）
- [ ] 如使用测量方法，描述测量标准

### Item 2c. 得气反应
- [ ] 是否追求得气（酸、麻、胀、重）
- [ ] 得气手法描述

### Item 2d. 针刺手法
- [ ] 进针手法（快速进针/缓慢进针）
- [ ] 补泻手法（捻转补泻、提插补泻、烧山火、透天凉等）
- [ ] 手法刺激参数（频率、幅度）

### Item 2e. 留针时间
- [ ] 每次治疗的留针时间（分钟）
- [ ] 留针期间是否行针

### Item 2f. 针具规格
- [ ] 针具品牌和型号
- [ ] 针具长度（mm）
- [ ] 针具直径（mm）

## 3. 治疗方案 (Treatment Regimen)

### Item 3a. 治疗频次
- [ ] 每周治疗次数
- [ ] 治疗间隔

### Item 3b. 治疗疗程
- [ ] 总治疗次数
- [ ] 治疗总时长（周/月）

### Item 3c. 治疗周期
- [ ] 每次治疗的持续时间
- [ ] 疗程间的间隔（如有）

## 4. 其他干预措施 (Other Components of Treatment)

### Item 4a. 辅助干预
- [ ] 电针参数（频率、波形、强度）
- [ ] 艾灸类型（直接/间接、艾柱/艾条）
- [ ] 拔罐参数（留罐时间、部位）
- [ ] 耳穴贴压（穴位、材料、按压频次）
- [ ] 其他辅助手法（TDP灯、激光等）

### Item 4b. 合并治疗
- [ ] 各组允许的合并治疗
- [ ] 禁止的合并治疗
- [ ] 合并治疗的记录方法

## 5. 治疗师背景 (Treatment Provider)

### Item 5a. 治疗师资质
- [ ] 从业年限
- [ ] 学历和资质证书
- [ ] 专业培训背景
- [ ] 每位治疗师的年均治疗量

## 6. 对照组 (Control or Comparator)

### Item 6a. 对照类型
- [ ] 对照组类型：
  - [ ] 假针刺（Sham acupuncture）— 浅刺/非穴位/Streitberger 针
  - [ ] 安慰针（Non-penetrating needle）
  - [ ] 最小针灸（Minimal acupuncture）— 非穴位浅刺
  - [ ] 常规治疗（Usual care）
  - [ ] 阳性对照（Active comparator）
  - [ ] 等待名单（Waitlist）

### Item 6b. 对照细节
- [ ] 对照组的具体操作描述
- [ ] 对照组的穴位选择（如适用）
- [ ] 对照组的治疗频次和疗程（与治疗组一致？）
- [ ] 盲法验证方法（如患者对组别的猜测）
