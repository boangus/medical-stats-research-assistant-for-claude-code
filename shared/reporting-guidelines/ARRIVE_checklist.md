# ARRIVE 报告规范检查清单

> 适用于动物实验研究
> 基于 ARRIVE 2.0 指南（Percie du Sert N, et al. PLOS Biology 2020;18:e3000410）

---

## 摘要部分

### 1. 标题和摘要
- [ ] 标题中包含物种和主要发现
- [ ] 结构化摘要包含：背景、目的、方法、结果、结论

## 引言部分

### 2. 背景和目的
- [ ] 说明研究背景和科学问题
- [ ] 明确研究目的和假设
- [ ] 说明动物模型的合理性

## 方法部分

### 3. 研究设计
- [ ] 描述研究设计（平行/交叉/析因等）
- [ ] 说明实验时间框架
- [ ] 说明随机化方法

### 4. 动物信息
- [ ] 描述物种、品系、性别、年龄、体重
- [ ] 说明动物来源和饲养条件
- [ ] 报告动物健康状态

### 5. 样本量
- [ ] 说明样本量计算方法
- [ ] 报告样本量依据（功率分析/文献参考）
- [ ] 说明是否进行样本量充足性评估

### 6. 随机化和盲法
- [ ] 说明随机化方法
- [ ] 说明盲法实施（评估者/操作者）
- [ ] 说明随机序列生成方法

### 7. 实验干预
- [ ] 描述干预措施详细方法
- [ ] 说明干预时间点和持续时间
- [ ] 报告干预实施细节（剂量/频率/途径）

### 8. 对照措施
- [ ] 描述对照组处理
- [ ] 说明对照类型（阴性/阳性/安慰剂）
- [ ] 报告对照处理细节

### 9. 结局测量
- [ ] 描述结局指标和测量方法
- [ ] 说明测量时间点
- [ ] 报告测量工具和参数

### 10. 统计分析
- [ ] 描述统计分析方法
- [ ] 说明多重比较校正方法
- [ ] 说明效应量和置信区间报告

## 结果部分

### 11. 动物流程
- [ ] 报告动物流程图
- [ ] 说明纳入和排除原因
- [ ] 报告动物损失情况

### 12. 基线特征
- [ ] 报告动物基线特征（体重/健康状态）
- [ ] 比较各组基线特征
- [ ] 报告随机化效果

### 13. 主要结果
- [ ] 报告主要结局指标结果
- [ ] 报告效应量和置信区间
- [ ] 报告P值和统计检验结果

### 14. 次要结果
- [ ] 报告次要结局指标结果
- [ ] 报告探索性分析结果
- [ ] 报告安全性结果（如适用）

## 讨论部分

### 15. 主要发现
- [ ] 总结主要发现
- [ ] 与文献比较
- [ ] 讨论科学意义

### 16. 局限性
- [ ] 讨论研究局限性
- [ ] 说明潜在偏倚来源
- [ ] 讨论动物模型的外推性

### 17. 结论
- [ ] 总结主要结论
- [ ] 说明临床转化价值
- [ ] 提出未来研究方向

## 其他部分

### 18. 伦理声明
- [ ] 报告伦理审批信息
- [ ] 说明动物福利措施
- [ ] 报告3R原则实施（替代/减少/优化）

### 19. 资金和利益冲突
- [ ] 报告资金来源
- [ ] 报告利益冲突声明

### 20. 代码和数据可用性
- [ ] 提供数据可用性声明（如适用）
- [ ] 提供代码可用性声明（如适用）

## ARRIVE 2.0 特有检查项

### 必须报告的项目（ARRIVE Essential 10）
1. **研究设计**：实验设计详细描述
2. **样本量**：样本量计算和依据
3. **动物信息**：物种、品系、性别、年龄、体重
4. **饲养条件**：饲养环境和条件
5. **随机化**：随机化方法和实施
6. **盲法**：盲法实施和评估
7. **干预措施**：干预详细描述
8. **结局测量**：结局指标和测量方法
9. **统计分析**：统计方法和报告
10. **伦理声明**：伦理审批和动物福利

### 建议报告的项目（ARRIVE Guidelines 2.0）
1. **动物健康状态**：健康监测和疾病状态
2. **样本量依据**：功率分析或文献参考
3. **效应量报告**：效应量和置信区间
4. **多重比较校正**：多重比较处理方法
5. **数据可用性**：数据共享声明

## 常见问题

### Q1: 需要多少动物？
A: 取决于研究设计和效应量。通常需要功率分析确定。

### Q2: 需要伦理审批吗？
A: 是的，所有动物实验都需要伦理审批。

### Q3: 如何实施3R原则？
A: 替代（使用非动物方法）、减少（优化样本量）、优化（改善动物福利）。
