# STROBE 报告规范检查清单

> 适用于观察性研究（队列、病例对照、横断面）
> 基于 STROBE 声明（2007）+ 扩展声明

---

## 摘要部分

### 1. 标题和摘要
- [ ] 标题中包含研究设计（队列/病例对照/横断面）
- [ ] 结构化摘要包含：背景、目的、方法、结果、结论
- [ ] 摘要中报告主要结局指标及效应量+95%CI

## 引言部分

### 2. 背景和目的
- [ ] 说明科学背景和 rationale
- [ ] 明确研究目的和假设
- [ ] 如适用，说明研究设计（嵌套研究等）

## 方法部分

### 3. 研究设计
- [ ] 描述研究设计（队列/病例对照/横断面/巢式病例对照）
- [ ] 说明研究时间框架和地点
- [ ] 描述研究人群的来源

### 4. 参与者
- [ ] 描述纳排标准（队列：暴露组和对照组；病例对照：病例和对照）
- [ ] 描述来源和招募方式
- [ ] 描述随访方法和时间（队列研究）
- [ ] 描述参与者流程图（如适用）

### 5. 变量
- [ ] 明确定义暴露因素（队列）/主要暴露/选择因素（病例对照）
- [ ] 明确定义结局指标
- [ ] 明确定义效应修饰因子和混杂因素
- [ ] 说明变量的测量方法和时间点

### 6. 数据来源/测量
- [ ] 描述各变量的数据来源
- [ ] 描述测量方法的详细信息
- [ ] 描述盲法评估（如适用）
- [ ] 描述暴露和结局的评估质量（如适用）

### 7. 偏倚
- [ ] 描述选择偏倚的来源和控制方法
- [ ] 描述信息偏倚的来源和控制方法
- [ ] 描述混杂的来源和控制方法

### 8. 研究 size
- [ ] 描述样本量计算方法（如适用）
- [ ] 说明精度估计（如适用）

### 9. 统计方法
- [ ] 描述主要结局的统计方法
- [ ] 描述分组变量和混杂控制方法
- [ ] 描述亚组分析和交互作用分析
- [ ] 描述缺失数据处理方法
- [ ] 描述敏感性分析方法
- [ ] 描述所有统计检验的显著性水平

## 结果部分

### 10. 参与者
- [ ] 报告各阶段参与者人数（筛选→纳入→分析）
- [ ] 报告随访情况和失访率（队列研究）
- [ ] 报告各组间的差异（如适用）

### 11. 描述性数据
- [ ] 报告各组参与者的基线特征
- [ ] 报告各组的暴露和结局发生率
- [ ] 描述数据的时间特征（如适用）

### 12. 结局数据
- [ ] 报告主要结局：效应量 + 95%CI
- [ ] 报告次要结局：效应量 + 95%CI
- [ ] 报告绝对和相对效应量
- [ ] 以适当方式呈现结果（如生存曲线、累积发生率等）

### 13. 其他分析
- [ ] 报告亚组分析和交互作用分析结果
- [ ] 报告敏感性分析结果
- [ ] 报告校正后的结果

## 讨论部分

### 14. 关键结果
- [ ] 概述主要发现
- [ ] 讨论主要发现的临床意义

### 15. 局限性
- [ ] 讨论偏倚来源（选择、信息、混杂）
- [ ] 讨论缺失数据的潜在影响
- [ ] 讨论测量误差的潜在影响
- [ ] 讨论残留混杂的可能性

### 16. 解读
- [ ] 考虑结果的一致性和可推广性
- [ ] 考虑其他相关证据
- [ ] 讨论临床意义 vs 统计显著性
- [ ] 讨论因果关系的强度（如适用）

## 其他

### 17. 可推广性
- [ ] 讨论结果的可推广性（外部效度）
- [ ] 讨论目标人群的代表性

### 18. 资金
- [ ] 描述资金来源和资助者角色

---

## 关键提示

1. **混杂控制**: 必须明确描述混杂因素的识别和控制方法（多变量调整/PSM/IPTW等），不能仅写"校正了混杂"
2. **缺失数据**: 必须描述缺失数据的比例和处理方法，不能静默排除
3. **敏感性分析**: 观察性研究必须包含至少1项敏感性分析，评估未测量混杂的影响
4. **E-value**: 当声称因果关系时，建议报告 E-value，量化未测量混杂的潜在影响
5. **暴露和结局定义**: 必须清晰、可重复，不能模糊定义
