# STROBE-ME 分子流行病学扩展报告规范

> 适用于分子流行病学观察性研究
> 基于 **STROBE** + **Molecular Epidemiology Extension**
> 参考: Gallo V, et al. (2011) STROBE-ME
> 更新日期: 2026-06-24

---

## Title and Abstract

### Item 1. 标题和摘要
- [ ] 标题中标识研究设计和分子标志物
- [ ] 摘要包含生物样本和标志物信息

## Introduction

### Item 2. 背景/原理
- [ ] 科学背景和分子机制假说

### Item 3. 目的
- [ ] 明确研究目的（暴露生物标志物、效应标志物等）

## Methods

### Item 4. 研究设计
- [ ] 描述研究设计
- [ ] 说明使用生物样本的时间框架

### Item 5. 研究设置
- [ ] 研究机构和时间

### Item 6. 参与者
- [ ] 纳入/排除标准
- [ ] 样本量确定依据

### Item 7. 变量
- [ ] 结局、暴露、混杂、效应修饰变量
- [ ] **🆕 分子标志物变量的详细描述**

### Item 8. 生物样本 🆕 **STROBE-ME**
- [ ] **样本采集**:
  - [ ] 采集时间（相对于疾病诊断/暴露的时间点）
  - [ ] 采集条件（禁食、时间、体位）
  - [ ] 采集方法（静脉穿刺、活检、尿液收集等）
- [ ] **样本处理**:
  - [ ] 处理时间（采集到处理的间隔）
  - [ ] 处理方法（离心参数、分装方法）
  - [ ] 添加剂/抗凝剂类型
- [ ] **样本存储**:
  - [ ] 存储温度（-80°C/-20°C/4°C）
  - [ ] 存储容器（冻存管类型）
  - [ ] 冻融次数记录
  - [ ] 存储时长

### Item 9. 实验室方法 🆕 **STROBE-ME**
- [ ] **检测方法**:
  - [ ] 方法名称（ELISA、LC-MS/MS、PCR、测序等）
  - [ ] 试剂盒品牌和货号
  - [ ] 检测仪器型号
- [ ] **质量控制**:
  - [ ] 批内/批间变异系数 (CV)
  - [ ] 检测限 (LOD) 和定量限 (LOQ)
  - [ ] 标准曲线和线性范围
  - [ ] 内部质控和外部质控
  - [ ] 重复检测率
- [ ] **盲法**:
  - [ ] 实验室人员是否对临床信息设盲
  - [ ] 样本编码和解盲流程

### Item 10. 统计方法
- [ ] 标志物的分类方法（连续/四分位/临床切点）
- [ ] 批效应校正方法（如有）
- [ ] 多重比较校正（如有多个标志物）

## Results

### Item 15. 参与者流程
- [ ] 样本流程图（采集→处理→存储→检测→分析）

### Item 16. 描述性数据
- [ ] 生物样本的基本特征（采集时间、存储时长、质量指标）

### Item 17. 结局数据
- [ ] 分子标志物的分布特征（均值/中位数/范围）
- [ ] 按组分层的标志物水平
