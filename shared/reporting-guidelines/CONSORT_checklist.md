# CONSORT 报告规范检查清单

> 适用于随机对照试验（RCT）
> 基于 **CONSORT 2025 声明**（Lancet/BMJ/JAMA 2025 发布）+ 2019 扩展（非药物干预）
> 更新日期: 2026-06-13

---

## 摘要部分

### 1. 标题和摘要
- [ ] 标题中包含"随机"或"randomised/randomized"
- [ ] 结构化摘要包含：背景、目的、方法、结果、结论
- [ ] 摘要中报告主要结局指标及效应量+95%CI

## 引言部分

### 2. 背景和目的
- [ ] 说明科学背景和 rationale
- [ ] 明确研究目的和假设

## 方法部分

### 3. 试验设计
- [ ] 描述试验设计（平行/交叉/析因等）
- [ ] 说明随机化比例
- [ ] 说明是否开放/单盲/双盲
- [ ] **[新增2025]** 说明是否为自适应设计，若是则描述适应规则

### 4. 参与者
- [ ] 描述合格标准
- [ ] 描述来源和招募方式
- [ ] 描述设置和地点
- [ ] **[新增2025]** 描述参与者多样性特征（年龄、性别、种族/民族、社会经济状况）
- [ ] **[新增2025]** 说明为促进多样性采取的措施（如有）

### 5. 干预措施
- [ ] 详细描述各组干预措施
- [ ] 描述实际实施情况（依从性）

### 6. 结局指标
- [ ] 明确定义主要和次要结局指标
- [ ] 说明任何衍生或复合指标
- [ ] 说明指标的测量方法和时间点

### 7. 样本量
- [ ] 描述样本量计算方法
- [ ] 报告预期效应量、α、Power
- [ ] 说明期中分析规则（如有）

### 8. 随机化
- [ ] 描述随机序列生成方法
- [ ] 描述分配隐藏机制
- [ ] 描述实施随机化的人员

### 8a. 盲法
- [ ] 描述盲法实施（谁被盲：参与者、干预实施者、结局评估者、统计分析师）
- [ ] 描述安慰剂/模拟干预（如有）

### 9. 统计方法
- [ ] 描述主要结局的统计方法
- [ ] 描述次要结局的统计方法
- [ ] 描述亚组分析和敏感性分析
- [ ] 描述缺失数据处理方法
- [ ] 描述ITT/PP分析策略
- [ ] **[新增2025]** 提供统计分析计划 (SAP) 的获取途径或引用

### 9a. 数据共享
- [ ] **[新增2025]** 说明个体参与者数据 (IPD) 的获取途径和条件
- [ ] **[新增2025]** 提供数据共享声明

## 结果部分

### 10. 参与者流程
- [ ] 提供 CONSORT 流程图
- [ ] 报告各阶段人数（随机化→分配→随访→分析）
- [ ] 报告失访和排除原因

### 11. 募集
- [ ] 报告招募期和随访期

### 12. 基线数据
- [ ] 以表格呈现各组基线特征
- [ ] 包括人口学和临床特征

### 13. 结局和估计
- [ ] 报告主要结局：效应量 + 95%CI + P值
- [ ] 报告次要结局：效应量 + 95%CI
- [ ] 报告绝对和相对效应量
- [ ] 报告其他分析结果（亚组、敏感性）

### 14. 副作用
- [ ] 报告各组不良事件发生率
- [ ] 报告严重不良事件
- [ ] **[新增2025]** 按系统器官分类报告不良事件

## 讨论部分

### 15. 局限性

## 关键提示

1. **随机化方法**: 必须详细描述随机序列生成和分配隐藏，不能仅写"随机分组"
2. **样本量论证**: 必须提供具体参数（效应量、α、Power），不能仅写"经计算所需样本量"
3. **主要结局**: 必须报告效应量 + 95%CI，不能仅报告P值
4. **ITT分析**: 首选ITT分析，PP分析作为敏感性分析
5. **流程图**: CONSORT 流程图是强制要求
6. **[新增2025] AI 披露**: 使用 MSRA 等 AI 辅助工具需在论文中明确声明
7. **[新增2025] 数据共享**: 建议提供 IPD 获取途径（如 Dryad、Zenodo 链接）
8. **[新增2025] 多样性**: 报告参与者多样性特征以评估结果的可推广性
