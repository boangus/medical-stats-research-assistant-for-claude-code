# CONSORT 报告规范检查清单

> 适用于随机对照试验（RCT）
> 基于 CONSORT 2010 声明 + 2019 扩展（非药物干预）

---

## 摘要部分

### 1. 标题和摘要
- [ ] 标题中包含"随机"或"randomised/randomized"
- [ ] 结构化摘要包含：背景、目的、方法、结果、结论
- [ ] 摘要中报告主要结局指标及效应量+95%CI

## 引言部分

### 2. 背景和目的
- [ ] 说明科学背景和 rationale
- [ ] 明确研究目的和假设

## 方法部分

### 3. 试验设计
- [ ] 描述试验设计（平行/交叉/析因等）
- [ ] 说明随机化比例
- [ ] 说明是否开放/单盲/双盲

### 4. 参与者
- [ ] 描述合格标准
- [ ] 描述来源和招募方式
- [ ] 描述设置和地点

### 5. 干预措施
- [ ] 详细描述各组干预措施
- [ ] 描述实际实施情况（依从性）

### 6. 结局指标
- [ ] 明确定义主要和次要结局指标
- [ ] 说明任何衍生或复合指标
- [ ] 说明指标的测量方法和时间点

### 7. 样本量
- [ ] 描述样本量计算方法
- [ ] 报告预期效应量、α、Power
- [ ] 说明期中分析规则（如有）

### 8. 随机化
- [ ] 描述随机序列生成方法
- [ ] 描述分配隐藏机制
- [ ] 描述实施随机化的人员

### 9. 统计方法
- [ ] 描述主要结局的统计方法
- [ ] 描述次要结局的统计方法
- [ ] 描述亚组分析和敏感性分析
- [ ] 描述缺失数据处理方法
- [ ] 描述ITT/PP分析策略

## 结果部分

### 10. 参与者流程
- [ ] 提供 CONSORT 流程图
- [ ] 报告各阶段人数（随机化→分配→随访→分析）
- [ ] 报告失访和排除原因

### 11. 募集
- [ ] 报告招募期和随访期

### 12. 基线数据
- [ ] 以表格呈现各组基线特征
- [ ] 包括人口学和临床特征

### 13. 结局和估计
- [ ] 报告主要结局：效应量 + 95%CI + P值
- [ ] 报告次要结局：效应量 + 95%CI
- [ ] 报告绝对和相对效应量
- [ ] 报告其他分析结果（亚组、敏感性）

### 14. 副作用
- [ ] 报告各组不良事件发生率
- [ ] 报告严重不良事件

## 讨论部分

### 15. 局限性
- [ ] 讨论试验局限性（偏倚来源）
- [ ] 讨论外部效度（可推广性）
- [ ] 讨论实施偏倚（依从性问题）

### 16. 解读
- [ ] 考虑结果的一致性和可推广性
- [ ] 考虑其他相关证据
- [ ] 讨论临床意义 vs 统计显著性

## 其他

### 17. 可推广性
- [ ] 讨论试验结果的可推广性

### 18. 资金
- [ ] 描述资金来源和资助者角色
- [ ] 描述资助者是否参与数据收集/分析/报告

---

## 关键提示

1. **随机化方法**: 必须详细描述随机序列生成和分配隐藏，不能仅写"随机分组"
2. **样本量论证**: 必须提供具体参数（效应量、α、Power），不能仅写"经计算所需样本量"
3. **主要结局**: 必须报告效应量 + 95%CI，不能仅报告P值
4. **ITT分析**: 首选ITT分析，PP分析作为敏感性分析
5. **流程图**: CONSORT流程图是强制要求
