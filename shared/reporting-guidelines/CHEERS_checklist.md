# CHEERS 2022 卫生经济学评估报告规范检查清单

> 适用于卫生经济学评估研究（成本效益、成本效用、成本效益分析等）
> 基于 **CHEERS 2022 声明**（Husereau et al., Value Health 2022; BMC Med 2022）
> 参考: [EQUATOR Network](https://www.equator-network.org/reporting-guidelines/cheers/)
> 28 项官方清单结构
> 更新日期: 2026-06-15

> **IRON RULES**:
> - 本清单严格对齐 CHEERS 2022 官方 28 项编号，不得自创编号体系
> - 新增项（相比 2013 版）以 **🆕 2022** 标注
> - 参考: shared/statistics-methods/ch40-cost-effectiveness.md（方法学）；本清单（报告规范）互补使用
> - 参考: shared/reporting-guidelines/CONSORT_checklist.md / STROBE_checklist.md（基础研究报告规范）

---

## Title

### Item 1. 标题
- [ ] 标题中标识为经济评估研究
- [ ] 说明评估类型（成本效益分析/成本效用分析/成本效益分析/预算影响分析）

---

## Abstract

### Item 2. 摘要
- [ ] 结构化摘要包含研究目的、方法、结果、结论
- [ ] 报告增量成本效果比（ICER）或净收益
- [ ] 报告不确定性分析结果（如置信区间、概率敏感性分析结果）

---

## Introduction

### Item 3. 背景和目的
- [ ] 说明研究背景和决策问题
- [ ] 明确研究视角（社会/卫生系统/患者/支付方）
- [ ] 说明研究目的和目标受众

---

## Methods

### Item 4. 研究人群
- [ ] 描述目标人群特征（年龄、性别、疾病状态）
- [ ] 说明纳入/排除标准
- [ ] 🆕 **2022** 说明健康公平性考量（如适用）

### Item 5. 比较方案
- [ ] 明确各比较方案（干预 vs 对照）
- [ ] 说明对照选择理由

### Item 6. 视角
- [ ] 明确声明评估视角
- [ ] 说明视角选择理由
- [ ] 说明成本和效果的纳入范围是否与视角一致

### Item 7. 时间范围
- [ ] 说明评估的时间范围
- [ ] 说明时间范围选择的理由（如疾病自然史、干预效果持续时间）

### Item 8. 折现率
- [ ] 报告成本和效果的折现率
- [ ] 说明折现率选择理由
- [ ] 如未折现，说明理由

### Item 9. 🆕 **2022** 评估日期
- [ ] 说明评估的时间段（数据收集、分析、报告日期）

### Item 10. 开放获取声明
- [ ] 🆕 **2022** 说明研究方案、数据、分析代码的可获取性

---

## Methods — 评估方法

### Item 11. 评估类型
- [ ] 明确评估类型（成本效益分析/成本效用分析/成本效益分析/预算影响分析）
- [ ] 说明结局指标选择理由（如 QALYs、DALYs、生命年）

### Item 12. 比较方案
- [ ] 说明是否与标准治疗或无干预比较
- [ ] 说明比较方案选择的合理性

### Item 13. 🆕 **2022** 患者和公众参与
- [ ] 说明患者/公众在研究设计、实施或报告中的参与方式
- [ ] 无 PPI 时须明确声明

---

## Methods — 模型结构

### Item 14. 模型类型
- [ ] 说明决策分析模型类型（决策树/Markov/microsimulation/分区生存模型等）
- [ ] 说明模型选择理由
- [ ] 描述模型结构（健康状态、转移概率）

### Item 15. 模型假设
- [ ] 列出所有关键假设
- [ ] 说明假设的证据来源
- [ ] 说明假设的合理性

### Item 16. 🆕 **2022** 循证来源
- [ ] 说明模型参数的证据来源（文献/数据库/专家意见）
- [ ] 说明证据检索策略（如适用）
- [ ] 说明参数选择理由

---

## Methods — 数据来源

### Item 17. 成本数据
- [ ] 说明成本数据来源（数据库/财务记录/文献）
- [ ] 说明成本分类方法（直接医疗/直接非医疗/间接成本）
- [ ] 报告成本价格年份和货币单位
- [ ] 说明成本数据的测量方法

### Item 18. 效果数据
- [ ] 说明效果数据来源
- [ ] 报告效果测量方法（如生存分析、生活质量测量工具）
- [ ] 说明效果数据的外推方法（如参数生存模型）

### Item 19. 🆕 **2022** 偏倚评估
- [ ] 说明对输入数据偏倚风险的评估方法（如 PROBAST、RoB 2）
- [ ] 说明如何处理输入数据的偏倚

---

## Methods — 不确定性分析

### Item 20. 单因素敏感性分析
- [ ] 说明单因素敏感性分析的参数选择
- [ ] 报告参数范围及其来源
- [ ] 以龙卷风图展示结果

### Item 21. 概率敏感性分析
- [ ] 说明概率敏感性分析方法（Monte Carlo 模拟）
- [ ] 报告参数分布类型及依据
- [ ] 以成本效果可接受曲线（CEAC）展示结果
- [ ] 报告各方案最优的概率

### Item 22. 🆕 **2022** 亚组分析
- [ ] 说明亚组分析的预设/事后性质
- [ ] 报告亚组分析结果及其对决策的影响

---

## Methods — 伦理和公平性

### Item 23. 🆕 **2022** 伦理
- [ ] 说明研究是否涉及伦理问题（如数据使用、人群公平性）
- [ ] 说明伦理审批情况（如适用）

---

## Results

### Item 24. 基线特征
- [ ] 报告目标人群基线特征
- [ ] 报告各比较方案的基线差异

### Item 25. 成本和效果
- [ ] 报告各方案的总成本和总效果
- [ ] 报告增量成本效果比（ICER）或净收益（INB）
- [ ] 报告各成本和效果组成部分

### Item 26. 不确定性分析结果
- [ ] 报告单因素敏感性分析结果
- [ ] 报告概率敏感性分析结果
- [ ] 报告成本效果可接受曲线（CEAC）
- [ ] 报告确定性分析（scenario analysis）结果（如有）

### Item 27. 亚组分析结果
- [ ] 报告亚组分析结果
- [ ] 说明亚组间差异是否具有统计学和临床意义

---

## Discussion

### Item 28. 研究发现、局限性、普遍性
- [ ] 总结主要研究发现
- [ ] 讨论局限性（模型假设、数据质量、外推不确定性）
- [ ] 讨论结果的普遍性（generalizability）和可转移性（transferability）
- [ ] 🆕 **2022** 讨论公平性影响（如适用）
- [ ] 说明对决策者的意义
