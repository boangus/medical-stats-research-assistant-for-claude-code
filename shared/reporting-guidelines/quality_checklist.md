# 医学统计分析质量检查清单

## 一、研究设计检查

### 1.1 研究类型适用性
- [ ] 研究问题与研究类型匹配
- [ ] 研究设计能够回答研究问题
- [ ] 对照组设置合理
- [ ] 暴露/干预定义清晰

### 1.2 样本量
- [ ] 样本量计算有依据
- [ ] 考虑了失访率/脱落率
- [ ] 检验效能 (Power) ≥ 80%
- [ ] 显著性水平 (α) 明确

### 1.3 偏倚控制
- [ ] 选择偏倚评估
- [ ] 信息偏倚评估
- [ ] 混杂因素识别
- [ ] 混杂控制方法适当

## 二、数据质量检查

### 2.1 数据完整性
- [ ] 缺失数据比例报告
- [ ] 缺失数据模式分析
- [ ] 缺失数据处理方法合理
- [ ] 完整案例分析 vs 多重插补

### 2.2 数据准确性
- [ ] 逻辑检查通过
- [ ] 范围检查通过
- [ ] 异常值处理合理
- [ ] 数据清理过程记录（建议使用 shared/reproducibility/pipeline_auditor.py 自动生成审计日志）

### 2.3 变量定义
- [ ] 所有变量定义清晰
- [ ] 主要结局指标明确
- [ ] 测量方法标准化
- [ ] 分类变量类别互斥且完备

## 三、统计方法检查

### 3.1 方法选择
- [ ] 方法适合数据类型
- [ ] 方法适合研究设计
- [ ] 考虑了数据相关性/聚类
- [ ] 考虑了重复测量

### 3.2 假设检验
- [ ] 正态性检验 (如需要)
- [ ] 方差齐性检验 (如需要)
- [ ] 比例风险假设检验 (生存分析)
- [ ] 线性关系检验 (线性回归)

### 3.3 多重比较
- [ ] 多重比较识别
- [ ] 校正方法应用 (Bonferroni, FDR等)
- [ ] 主要/次要终点区分
- [ ] 事后检验合理性

## 四、结果报告检查

### 4.1 描述性统计
- [ ] Table 1 完整
- [ ] 连续变量报告均值±SD或中位数(IQR)
- [ ] 分类变量报告频数(百分比)
- [ ] 组间比较 p 值报告

### 4.2 主要结果
- [ ] 效应量报告 (OR, HR, MD等)
- [ ] 95% 置信区间报告
- [ ] p 值精确值或范围
- [ ] 精确样本量 (分母) 报告

### 4.3 模型诊断
- [ ] 模型拟合度指标
- [ ] 残差分析 (如适用)
- [ ] 影响点检查
- [ ] 多重共线性检查

## 五、报告规范检查

### 5.1 CONSORT (RCT)
- [ ] 流程图完整
- [ ] 随机化方法描述
- [ ] 盲法实施描述
- [ ] 脱落/失访报告

### 5.2 STROBE (观察性研究)
- [ ] 研究设计描述
- [ ] 纳入排除标准
- [ ] 暴露/结局测量方法
- [ ] 潜在偏倚讨论

### 5.3 图表质量
- [ ] 图表自明性
- [ ] 坐标轴标签清晰
- [ ] 图例完整
- [ ] 统计信息完整 (误差线、样本量等)

## 六、解释与结论

### 6.1 统计显著性
- [ ] 避免"边缘显著"表述
- [ ] p 值正确解释
- [ ] 置信区间解释
- [ ] 临床意义讨论

### 6.2 因果推断
- [ ] 观察性研究避免因果表述
- [ ] 混杂因素讨论
- [ ] 反向因果考虑
- [ ] 效应修饰因子探索

### 6.3 局限性
- [ ] 统计局限性讨论
- [ ] 方法学局限性讨论
- [ ] 结果外推性讨论
- [ ] 未来研究方向

## 七、常见错误警示

### 严重错误 (必须修正)
- [ ] 方法完全不适合数据
- [ ] 样本量严重不足
- [ ] 主要假设严重违反
- [ ] 数据造假嫌疑

### 重要错误 (建议修正)
- [ ] 未校正多重比较
- [ ] 缺失数据处理不当
- [ ] 混杂控制不足
- [ ] 效应量未报告

### 轻微错误 (最好修正)
- [ ] 小数点位数不一致
- [ ] 术语使用不规范
- [ ] 图表美观度
- [ ] 格式细节

## 八、最终确认

- [ ] 所有分析可重复
- [ ] 代码/脚本保存
- [ ] 数据备份
- [ ] 版本控制
