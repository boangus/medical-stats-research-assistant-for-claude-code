# CONSORT-TCM 中医药扩展报告规范检查清单

> 适用于中医药随机对照试验
> 基于 **CONSORT 2025** + **CONSORT-TCM 扩展声明**
> 参考: Bian ZX, et al. (2013) CONSORT Extension for Reporting Randomised Controlled Trials of Herbal Interventions
> 更新日期: 2026-06-24

---

## Title and Abstract

### Item 1. 标题和摘要
- [ ] 标题中标识为随机试验 + 中医药干预类型（如"中药方剂"/"针灸"/"推拿"）
- [ ] 结构化摘要包含中医辨证分型信息

## Introduction

### Item 2. 背景和目的
- [ ] 中医理论依据（如经典文献引用、名医经验）
- [ ] 现代医学机制（如有）
- [ ] 明确研究目的和预设假设

## Methods

### Item 3. 试验设计
- [ ] 描述试验设计（平行/交叉/析因）
- [ ] 中医证候分层随机化（如适用）

### Item 5. 参与者
- [ ] **西医诊断标准**（如 ICD-11、临床指南）
- [ ] **中医辨证标准**（如《中药新药临床研究指导原则》证候分型）
- [ ] 纳入/排除标准中注明中医证候要求
- [ ] 招募来源和设置

### Item 6. 干预措施 🆕 **TCM**
- [ ] **中药方剂**:
  - [ ] 方剂名称和来源（经典方/经验方/自拟方）
  - [ ] 完整药物组成（药名、剂量、产地）—— 君臣佐使分列
  - [ ] 药材基原和质量标准（如《中国药典》标准）
  - [ ] 制备工艺（汤剂/颗粒/丸散）
  - [ ] 给药方式、剂量、频次、疗程
  - [ ] 安慰剂/模拟干预的制备方法（如使用外观一致的模拟药）
- [ ] **针灸干预**:
  - [ ] 穴位选择（主穴+配穴，含经络归属）
  - [ ] 取穴依据（经典文献/临床经验/指南推荐）
  - [ ] 操作手法（进针深度、得气、补泻手法）
  - [ ] 留针时间、频次、疗程
  - [ ] 针具规格（品牌、型号、长度、直径）
  - [ ] 操作者资质（从业年限、培训背景）
- [ ] **其他中医干预**（推拿/拔罐/艾灸/耳穴等）的标准化描述

### Item 7. 结局指标
- [ ] 主要/次要结局的操作性定义
- [ ] **中医证候疗效评价**（如适用）:
  - [ ] 证候量化评分表（主症+次症分级量化）
  - [ ] 证候疗效判定标准（痊愈/显效/有效/无效）

### Item 13a. 盲法
- [ ] 盲法实施：谁被盲（参与者/实施者/结局评估者/统计分析师）
- [ ] 中药安慰剂的制备和验证（外观、气味、口感一致性）
- [ ] 针灸假针对照的设计（浅刺/非穴位/不刺入）

### Item 14. 统计方法
- [ ] 主要/次要结局的统计方法
- [ ] 中医证候积分的统计方法（Wilcoxon signed-rank / McNemar）
- [ ] 亚组分析（按证候分型分层）
- [ ] 缺失数据处理方法

## Results

### Item 15. 参与者流程
- [ ] CONSORT 流程图（含中医证候筛选环节）
- [ ] 各证候分型的入组人数

### Item 17. 基线数据
- [ ] 各组基线特征表（西医指标 + 中医证候分布）
- [ ] 各证候分型的基线特征

### Item 19. 结局和估计
- [ ] 主要结局：效应量 + 95%CI + 精确 P 值
- [ ] **中医证候疗效**：各组证候积分变化、证候有效率
- [ ] 按证候分型分层的结果（如适用）

## Discussion

### Item 23. 局限性
- [ ] 中医辨证标准化的局限性
- [ ] 中药质量控制的挑战
- [ ] 盲法实施的困难

### Item 24. 临床意义 vs 统计显著性
- [ ] 中医临床意义的讨论（证候改善 vs 西医指标改善）

## CONSORT-TCM 核心新增项速查

| Item | 新增内容 | 缺失判定 |
|------|---------|---------|
| 6-TCM | 中药方剂完整组成（君臣佐使） | **核心缺失→不合规** |
| 6-TCM | 针灸干预 STRICTA 详情 | 针灸试验必查 |
| 6-TCM | 中药材质量标准 | 建议补充 |
| 7-TCM | 中医证候疗效评价 | 证候结局必查 |
| 13-TCM | 中药安慰剂/假针对照验证 | 有对照组时必查 |
| 17-TCM | 基线中医证候分布 | 建议补充 |
