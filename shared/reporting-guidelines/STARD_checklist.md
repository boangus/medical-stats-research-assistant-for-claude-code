## 2×2 列联表模板

```
                    金标准
                    阳性    阴性    合计
待评估试验  阳性    TP      FP      TP+FP
            阴性    FN      TN      FN+TN
            合计    TP+FN   FP+TN   N
```

**计算公式**：
- 敏感度 = TP / (TP + FN)
- 特异度 = TN / (FP + TN)
- PPV = TP / (TP + FP)
- NPV = TN / (FN + TN)
- LR+ = 敏感度 / (1 - 特异度)
- LR- = (1 - 敏感度) / 特异度
- DOR = LR+ / LR-

## 与 QUADAS-2 的互补关系

| 维度 | STARD（报告规范） | QUADAS-2（偏倚评估） |
|------|-------------------|---------------------|
| 目的 | 研究是否报告了所有必要信息 | 研究是否存在偏倚风险 |
| 时机 | 撰写/审稿时 | 系统综述/Meta分析时 |
| 判断 | 符合/不符合报告条目 | Low/High/Unclear |
| 对应 | Item 15（2×2 列联表）→ D4 流程 | Item 16（准确性指标）→ D2 待评估试验 |

---

## 与 TRIPOD+AI 的区别

| 特征 | STARD（诊断准确性） | TRIPOD+AI（预测模型） |
|------|---------------------|------------------------|
| 适用研究 | 单个诊断试验准确性 | 临床预测模型开发/验证 |
| 核心指标 | 敏感度/特异度 | AUC/校准度 |
| 结果类型 | 二分类（有病/无病） | 连续概率 |
| 阈值 | 单个或多个诊断阈值 | 概率阈值（决策曲线） |
| 偏倚评估 | QUADAS-2 | PROBAST |
