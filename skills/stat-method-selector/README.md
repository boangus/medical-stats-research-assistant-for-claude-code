# 统计方法选择器 (stat-method-selector)

一个运行在 **Claude Code** 上的 AI 技能，帮助科研人员从"不知道用什么统计方法"到"就是这个方法"。

## 这是什么？

基于 **53 篇已发表方法学论文** 构建的统计方法决策树，覆盖 **15 个研究目标分支、112 个统计方法叶子节点**。支持两种输入方式：

- **问卷式引导** — 一步步问你，5-8 步定位到推荐方法
- **自然语言描述** — 直接描述研究场景，自动解析并匹配

每次推荐附带：SPSS 操作路径、R 代码、效应量标准、文献依据，以及贝叶斯替代方案。

## 快速开始

### 方式一：Claude Code Skill（推荐）

将整个文件夹放入 Claude Code 的技能目录：

```
.claude/skills/stat-method-selector/
```

然后在 Claude Code 中说「帮我选统计方法」即可。

### 方式二：独立 Web 界面

```bash
# 启动本地服务器
start-server.bat
# 浏览器访问 http://localhost:8090
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `SKILL.md` | Skill 定义文件，包含完整的使用逻辑和输出格式 |
| `decision-tree.json` | 决策树数据（6-8层维度，112个叶子节点） |
| `references.md` | 53篇方法学论文的结构化摘要 |
| `index.html` | Web 版交互界面（独立使用） |
| `server.js` | 本地 HTTP 服务器 |
| `audit.py` | 决策树质量审计脚本 |

## 适用范围

侧重**医学与健康科学**领域。多数方法（t检验、ANOVA、回归等）在其他领域通用，但因果推断、诊断试验、生存分析等分支专门针对临床研究场景。
