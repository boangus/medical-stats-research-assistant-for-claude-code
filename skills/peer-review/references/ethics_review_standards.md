# Ethics Review Standards

> 伦理审查标准 + ICMJE 伦理要求 + 赫尔辛基宣言要点
> 适用: 伦理审查员 | v0.9.0

---

## 1. ICMJE 伦理要求清单

### 1.1 伦理审批

| 检查项 | 要求 | 严重性 |
|--------|------|--------|
| IRB/伦理委员会批准声明 | 必须在方法部分明确声明批准机构和批准号 | CRITICAL |
| 审批范围 | 审批是否涵盖所有研究内容 | MAJOR |
| 审批有效期 | 研究是否在审批有效期内完成 | MAJOR |

### 1.2 知情同意

| 检查项 | 要求 | 严重性 |
|--------|------|--------|
| 书面知情同意 | 所有参与者签署知情同意书 | CRITICAL |
| 电子知情同意 | 如使用电子版，需说明安全性保障 | MAJOR |
| 知情同意豁免 | 如豁免，必须说明豁免理由和批准机构 | CRITICAL |
| 弱势群体 | 儿童/认知障碍者需法定代理人同意 | CRITICAL |

### 1.3 数据隐私

| 检查项 | 要求 | 严重性 |
|--------|------|--------|
| 去标识化 | 数据必须去标识化处理 | MAJOR |
| GDPR/HIPAA 合规 | 根据研究地区遵守相应法规 | MAJOR |
| 数据存储安全 | 说明数据存储位置和安全措施 | MINOR |

### 1.4 利益冲突

| 检查项 | 要求 | 严重性 |
|--------|------|--------|
| COI 声明 | 所有作者必须提交 ICMJE Disclosure Form | CRITICAL |
| 资金来源 | 完整声明所有资金来源及基金号 | MAJOR |
| 赞助者角色 | 说明赞助者在研究各环节的角色 | MAJOR |

### 1.5 临床试验注册

| 检查项 | 要求 | 严重性 |
|--------|------|--------|
| 注册号 | 首例入组前完成注册，必须报告注册号 | CRITICAL |
| 注册平台 | ClinicalTrials.gov / ISRCTN / WHO ICTRP | — |
| 注册与报告一致性 | 主要结局指标必须与注册一致 | MAJOR |

## 2. 赫尔辛基宣言核心原则

### 基本原则

| 原则 | 要求 |
|------|------|
| **尊重个人** | 受试者自主权、知情同意权、退出权 |
| **受益最大化/风险最小化** | 研究风险必须合理地预期受益 |
| **公正性** | 研究负担和受益的公平分配 |
| **独立审查** | 研究必须经过独立的伦理审查 |
| **科学有效性** | 研究必须具备科学合理性（伦理审查的前提） |

### 对弱势群体的保护

| 群体 | 额外保护要求 |
|------|-------------|
| 儿童 | 法定代理人同意 + 儿童适当赞同 (assent) |
| 孕妇 | 仅在对母婴的直接获益大于风险时 |
| 认知障碍者 | 法定代理人同意 + 尽可能获取本人赞同 |
| 囚犯 | 确保非胁迫参与 + 独立审查 |
| 经济困难者 | 不得因经济补偿诱导参与 |

### 研究结束后义务

| 义务 | 说明 |
|------|------|
| 结果通知 | 向参与者和公众报告研究结果 |
| 干预延续 | 证明有效的干预应在研究结束后继续提供 |
| 数据保存 | 研究数据保存期限要求 |

## 3. 不同研究类型的伦理特殊要求

### 3.1 观察性研究

| 要求 | 说明 |
|------|------|
| 知情同意 | 可申请豁免（如使用匿名二手数据），需说明理由 |
| 隐私保护 | 去标识化必须彻底，防止再识别 |
| 社区知情 | 涉及社区的研究需社区知情同意 |

### 3.2 诊断准确性研究

| 要求 | 说明 |
|------|------|
| 金标准风险 | 金标准检查必须在合理风险范围内 |
| 不确定结果 | 不确定/边缘结果的后续处理须告知参与者 |

### 3.3 预测模型研究

| 要求 | 说明 |
|------|------|
| 数据来源 | 必须说明数据来源和获取合法性 |
| 算法公平性 | 需讨论模型在不同人群中的公平性 |

### 3.4 系统综述

| 要求 | 说明 |
|------|------|
| 伦理审批 | 通常不需要（因不直接接触参与者） |
| 协议注册 | PROSPERO 注册 |

## 4. 伦理审查员审查流程

```
Step 1: 声明检查
  ├── IRB 声明是否完整？ → 缺失 = CRITICAL
  ├── 知情同意是否描述？ → 缺失 = CRITICAL
  └── 利益冲突是否声明？ → 缺失 = CRITICAL

Step 2: 内容审查
  ├── 赫尔辛基宣言合规性
  ├── 弱势群体保护（如适用）
  └── 数据隐私保护

Step 3: 特殊场景审查
  ├── 临床试验注册一致性
  ├── 多中心研究伦理
  └── 国际研究跨国伦理

Step 4: 评估报告
  └── 输出伦理审稿报告（遵循审稿报告模板）
```
