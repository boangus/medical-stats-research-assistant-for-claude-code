# Statistical Review Standards

> 统计审查标准 + 医学期刊统计报告要求
> 适用: 统计审稿人 | v0.9.0

---

## 1. 统计方法审查清单

### 1.1 方法选择合理性

| 检查项 | 要求 |
|--------|------|
| 参数 vs 非参数 | 必须说明选择依据（正态性检验/数据特征） |
| 模型假设验证 | 正态性、等方差、线性、独立性等假设必须验证 |
| 多重比较校正 | Bonferroni / FDR / 预注册分析计划，必须说明 |
| 缺失数据处理 | 缺失机制 (MCAR/MAR/MNAR) 判断 + 处理方法 |
| 敏感性分析 | 核心结论在不同分析策略下的稳健性 |

### 1.2 样本量与功效

| 检查项 | 要求 |
|--------|------|
| 先验功效分析 | 必须在研究设计阶段完成 |
| 效应量估计 | 基于文献或预试验的合理估计 |
| 检验水准 | α 通常为 0.05，需明确说明 |
| 检验功效 | 通常要求 ≥ 0.80 |
| 事后功效分析 | 不推荐，但需说明理由 |

### 1.3 结果报告

| 检查项 | 要求 |
|--------|------|
| 效应量 | 必须报告效应量 (Cohen's d / OR / HR / RR) |
| 置信区间 | 必须报告 95% CI（而非仅 p 值） |
| p 值格式 | P < 0.001 而非 P = 0.000 |
| 精确 p 值 | p > 0.001 时报告精确值 |
| 多重性 | 预先设定的亚组分析 vs 探索性分析需区分 |

## 2. 常见统计方法适用场景

| 研究目的 | 推荐方法 | 替代方法 | 注意事项 |
|----------|---------|---------|---------|
| 两组均值比较 | 独立 t 检验 | Mann-Whitney U | 需验证正态性和方差齐性 |
| 配对均值比较 | 配对 t 检验 | Wilcoxon 符号秩 | 需验证差值正态性 |
| 多组均值比较 | ANOVA | Kruskal-Wallis | 正态+方差齐 → ANOVA |
| 重复测量 | 重复测量 ANOVA / 混合效应模型 | GEE | 选择取决于研究问题 |
| 分类结局 | 卡方检验 / Fisher | logistic 回归 | 期望频数 <5 时用 Fisher |
| 生存分析 | Kaplan-Meier + Log-rank | Cox 回归 | 需验证比例风险假设 |
| 线性预测 | 线性回归 | 广义线性模型 | 检查残差图 |
| 二分类预测 | logistic 回归 | 决策树 / 随机森林 | 检查多重共线性 |
| 生存预测 | Cox 回归 | 随机生存森林 | 检查 PH 假设 |

## 3. 医学期刊统计报告要求

### 必须报告项目

| # | 项目 | 说明 |
|---|------|------|
| 1 | 统计方法 | 每个分析使用的具体方法名称 |
| 2 | 软件版本 | 统计软件及版本号 (R x.x.x / SAS x.x / SPSS x.x) |
| 3 | 显著性水平 | 预设的 α 水平及理由 |
| 4 | 双侧/单侧 | 检验方向及理由 |
| 5 | 样本量计算 | 功效分析依据、效应量估计 |
| 6 | 缺失数据 | 缺失数量、机制判断、处理方法 |
| 7 | 敏感性分析 | 分析策略变更对结果的影响 |
| 8 | 效应量 + CI | 而非仅报告 p 值 |

### 常见统计错误

| # | 错误 | 严重性 | 说明 |
|---|------|--------|------|
| 1 | 仅报告 p 值无效应量 | MAJOR | p < 0.05 不等于临床有意义 |
| 2 | 未验证模型假设 | CRITICAL | t 检验用于严重偏态数据 |
| 3 | 多重比较未校正 | MAJOR | 多次检验增加假阳性 |
| 4 | 缺失数据处理不当 | MAJOR | 完全案例分析可能引入偏倚 |
| 5 | 样本量不足无解释 | MAJOR | 功效 < 0.80 的研究结论不可靠 |
| 6 | 事后功效分析 | MINOR | 事后功效无统计学意义 |
| 7 | 混淆相关与因果 | CRITICAL | 观察性研究中的因果推断 |
| 8 | 比例风险假设未验证 | MAJOR | Cox 回归的前提条件 |

## 4. 统计审稿人评审视角

### 必须独立评估 (IR3)

| 维度 | 关注点 |
|------|--------|
| 方法正确性 | 统计方法是否适合研究问题和数据类型 |
| 假设验证 | 模型假设是否经过检验 |
| 分析完整性 | 是否遗漏必要的分析 |
| 报告透明度 | 是否充分描述统计方法和结果 |
| 可重复性 | 是否提供足够信息供他人重复分析 |

### 不属于统计审稿人职责

| 维度 | 负责人 |
|------|--------|
| 临床意义判断 | 临床专家 |
| 研究设计合理性 | 方法学审稿人 |
| 伦理合规性 | 伦理审查员 |
| 总体学术价值 | 主编 |
