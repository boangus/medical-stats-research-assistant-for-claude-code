# Review Criteria Framework

> 按论文类型区分的结构化评审标准
> 适用: 全部审稿人 | v0.9.0

---

## 1. 随机对照试验 (RCT)

### 核心评审要点
- **随机化**: 序列生成方法、分配隐藏机制、随机化基线平衡
- **盲法**: 受试者/干预实施者/结局评估者/统计分析者的盲法设置
- **主要结局**: 预注册一致性、主要/次要结局指标区分
- **样本量**: 先验功效分析、效应量估计依据
- **ITT 分析**: 意向性治疗原则遵守、偏离方案处理
- **不良事件**: SAE/AE 完整报告、因果关系评估

### 优先检查报告规范
- **CONSORT 2025** → 见 `src/shared/reporting-guidelines/consort/`
- 流程图完整性（随机→分配→随访→分析）
- 必须报告 CONSORT 清单项完成情况

## 2. 观察性研究 (队列/病例对照/横断面)

### 核心评审要点
- **研究设计**: 类型明确声明、时间框架、研究环境
- **参与者**: 纳入/排除标准、来源、招募方法
- **暴露与结局**: 变量定义、测量方法、时间关系
- **混杂控制**: 混杂因素识别、调整策略（匹配/分层/模型调整）
- **偏倚风险**: 选择偏倚、信息偏倚、失访偏倚的讨论
- **敏感性分析**: 排除特定人群/改变定义后的稳健性

### 优先检查报告规范
- **STROBE** → 见 `src/shared/reporting-guidelines/strobe/`
- 队列/病例对照/横断面各有专门清单项

---

## 3. 诊断准确性研究

### 核心评审要点
- **金标准**: 参考标准的合理性与独立性
- **受试者流程**: 纳入方式（连续/随机/病例对照）影响结果解读
- **诊断性能**: 敏感度、特异度、AUC 及 95% CI
- **似然比**: 阳性/阴性似然比的临床实用性
- **阈值**: 诊断阈值确定方法、不确定结果处理
- **亚组分析**: 不同人群/不同阈值下的性能差异

### 优先检查报告规范
- **STARD 2015** → 见 `src/shared/reporting-guidelines/stard/`

---

## 4. 系统综述与 Meta 分析

### 核心评审要点
- **方案注册**: PROSPERO 注册、协议发表
- **检索策略**: 数据库覆盖度、检索式完整性、时间范围
- **筛选流程**: 双人独立筛选、不一致解决机制
- **数据提取**: 提取表设计、提取者培训
- **偏倚评估**: RoB 2 / ROBINS-I / NOS / ROBIS 工具使用
- **证据质量**: GRADE 分级
- **定量合成**: 异质性检验 (I², Q 检验)、模型选择（固定/随机）、发表偏倚

### 优先检查报告规范
- **PRISMA 2020** → 见 `src/shared/reporting-guidelines/prisma/`
- PRISMA-S（检索报告）、PRISMA-IPD（个体患者数据）视适用性使用

---

## 5. 预测模型研究

### 核心评审要点
- **开发与验证**: 开发队列 vs 验证队列的独立性
- **预测因子**: 定义清晰、可临床获取
- **模型性能**: 区分度 (C-statistic/AUC)、校准度 (calibration plot, Brier score)
- **验证方法**: Bootstrap / 交叉验证 / 外部独立验证
- **可及性**: 预测方程/代码/工具是否公开
- **AI 透明度**: ML 模型的可解释性、训练数据描述

### 优先检查报告规范
- **TRIPOD+AI** → 见 `src/shared/reporting-guidelines/tripod/`

---

## 6. 病例报告

### 核心评审要点
- **患者信息**: 人口学与临床特征完整性
- **临床发现**: 体格检查与辅助检查结果
- **时间线**: 诊断与治疗的时间线清晰度
- **诊断评估**: 诊断方法与鉴别诊断逻辑
- **治疗与结局**: 干预措施与患者结局报告
- **教育意义**: 病例对临床实践的启示

### 优先检查报告规范
- **CARE** → 见 `src/shared/reporting-guidelines/care/`

---

## 通用评审维度 (所有类型适用)

| 维度 | 检查内容 |
|------|---------|
| **ICMJE 合规** | 署名、利益冲突、数据共享声明 |
| **伦理合规** | IRB 审批、知情同意、临床试验注册 |
| **统计方法** | 方法选择合理性、假设验证、敏感性分析 |
| **学术创新** | 相对于已有文献的增量贡献 |
| **临床意义** | 对临床实践的实际影响 |
