## 各期刊标题页特殊要求

| 期刊 | 标题字数 | 摘要类型 | 摘要词数 | 关键词数 | 特殊要求 |
|------|---------|---------|---------|---------|---------|
| NEJM | ≤96字符 | 结构化 | ≤250 | 无 | 需What's New Box |
| Lancet | ≤100字符 | 结构化 | ≤300 | 3-6 | 需Research in Context Box |
| JAMA | ≤150字符 | 结构化 | ≤350 | 无 | 需Key Points Box |
| BMJ | ≤90字符 | 结构化 | ≤300 | 无 | 需What is Already Known Box |
| Annals | ≤100字符 | 非结构化 | ≤250 | 无 | — |
| PLOS Medicine | ≤150字符 | 结构化 | ≤300 | 无 | 需Author Summary |
