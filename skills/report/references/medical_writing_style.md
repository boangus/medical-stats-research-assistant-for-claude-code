# 医学论文写作风格指南

> 本指南为 MSRA 论文写作模块（manuscript_drafter / mock_reviewer）提供医学论文的写作风格标准。
> 基于 ICMJE 推荐、AMA 手册、NEJM/Lancet/JAMA/BMJ 投稿规范编写。

## 核心原则

### 1. 证据层级意识
- 所有声明必须有证据支持，并标注证据等级
- 因果语言仅用于 RCT 结果，观察性研究使用关联语言
- 系统综述/Meta 分析的结论强度高于单个研究

### 2. 临床精确性
- 使用标准医学术语（首次出现时给出全称和缩写）
- 统计数据必须完整：效应量 + 95%CI + P 值
- 不得省略分母（样本量）和缺失值说明

### 3. 报告规范合规
- RCT: CONSORT 2025
- 观察性: STROBE
- 系统综述: PRISMA 2020
- 诊断准确性: STARD 2015
- 预测模型: TRIPOD+AI

### 4. 审慎表达
- 观察性研究不得使用"证明""导致""引起"等因果词汇
- 使用"提示""关联""与...相关"等关联词汇
- 局限性必须在讨论部分明确陈述

## 医学论文各部分写作风格

### 摘要 (Abstract)
```
结构: 背景 → 目的 → 方法 → 结果 → 结论
字数: 250-300 词（英文）/ 400-500 字（中文）
时态: 背景(现在) → 方法(过去) → 结果(过去) → 结论(现在)
语言: 简洁、信息密度高、无冗余修饰
```

### 引言 (Introduction)
```
结构: 漏斗形 — 背景 → 知识空白 → 研究目的
段落: 3-4 段
时态: 背景(现在) → 知识空白(现在完成) → 目的(现在)
要点:
- 第1段: 疾病负担和临床重要性
- 第2段: 现有证据和知识空白
- 第3段: 研究目的和假设
```

### 方法 (Methods)
```
结构: 研究设计 → 人群 → 变量 → 统计方法
时态: 全部过去时
要点:
- 研究设计: 明确标注（RCT/队列/病例对照/横断面）
- 研究人群: 纳入/排除标准、样本量计算
- 变量定义: 暴露/结局/协变量的明确定义
- 统计方法: 与 SAP 一致，标注软件版本
- 伦理审批: IRB 批准号、知情同意声明
```

### 结果 (Results)
```
结构: 描述性统计 → 主要分析 → 次要分析 → 敏感性分析
时态: 全部过去时
要点:
- 先描述基线特征（Table 1）
- 主要结局优先报告
- 效应量必须附 95%CI
- P 值精确到 3 位小数（P<0.001 统一格式）
- 不得在结果部分解释数据
```

### 讨论 (Discussion)
```
结构: 主要发现 → 与文献比较 → 机制解释 → 局限性 → 结论
时态: 主要发现(过去) → 比较(现在) → 结论(现在)
要点:
- 第1段: 重述主要发现（不重复数字）
- 第2-3段: 与已有文献比较（一致/不一致及可能原因）
- 第4段: 潜在机制解释
- 第5段: 局限性（至少 3 条）
- 最后段: 结论和临床意义
```

## 统计数据报告格式

### 效应量
```
风险比: HR = 1.23 (95% CI: 1.05-1.45)
比值比: OR = 2.1 (95% CI: 1.3-3.4)
相对危险度: RR = 0.8 (95% CI: 0.6-1.0)
均数差: MD = -3.2 (95% CI: -5.1 to -1.3)
标准化均数差: SMD = 0.45 (95% CI: 0.21-0.69)
```

### 描述性统计
```
正态分布: 23.5 ± 4.2 (均数 ± 标准差)
非正态分布: 45.2 (32.1, 58.3) (中位数 [Q1, Q3])
分类变量: 123 (45.6%) (频数 [百分比])
```

### P 值
```
精确值: P = 0.023
界限值: P < 0.001 (禁止 P = 0.000)
不显著: P = 0.15 (禁止 P = NS)
```

### 生存分析
```
中位生存: 24.5 个月 (95% CI: 18.3-30.7)
生存率: 5 年生存率 78.5% (95% CI: 72.1-84.9%)
风险比: HR = 0.72 (95% CI: 0.54-0.96), Log-rank P = 0.023
```

## 审慎表达（Hedging）

### 观察性研究必须使用的表达
| 避免 | 使用 |
|------|------|
| X 导致 Y | X 与 Y 相关 / X 与 Y 存在关联 |
| X 增加 Y 的风险 | X 与 Y 风险升高相关 |
| 证明 | 提示 / 支持 / 表明 |
| 因果关系 | 关联 / 相关性 |

### 结论强度限定
| 强度 | 表达 | 适用场景 |
|------|------|---------|
| 强 | 证实 / 确立 / 明确 | RCT 主要结局、Meta 分析一致性结论 |
| 中 | 提示 / 支持 / 表明 | 观察性研究主要分析、敏感性分析一致 |
| 弱 | 可能 / 或许 / 初步提示 | 探索性分析、亚组分析、样本量小 |

## 常见错误

### 统计报告错误
| 错误 | 正确 |
|------|------|
| P = 0.000 | P < 0.001 |
| CI 95% (1.2, 3.4) | 95% CI: 1.2-3.4 |
| HR 1.23 (1.05-1.45) | HR = 1.23 (95% CI: 1.05-1.45) |
| 中位生存 24.5 月 | 中位生存 24.5 个月 (95% CI: 18.3-30.7) |
| 有效率 80% | 有效率 80% (120/150) |

### 语言错误
| 错误 | 正确 |
|------|------|
| 本研究证明了... | 本研究提示... / 本研究结果支持... |
| 显著提高了 (without P) | 统计学显著提高 (P = 0.02) |
| 大多数患者 | 78.5% 的患者 (n = 156/199) |
| 近年来 | 2020-2025 年间 |
| 相关文献 | Chen et al. (2024) 和 Smith et al. (2023) 的研究 |
